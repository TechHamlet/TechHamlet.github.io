Mozilla Have Just Released Firefox 8!

Nov 8th, 2011 | Comments

Just now Mozilla officially released Firefox 8! As an answer for Chrome’s rapid release cycle, this year, Firefox have also decided to release new versions a lot faster than before. Actually, it’s very essential to keep up with the rapidly evolving world of technology! So, if you are ready, lets see whats new in this release of Mozilla Firefox!

The latest Firefox 8 will be focused more on security. Specially, on security threats caused by third-party plugins! Normally, there are two ways a plugin can get installed onto your browser :

  1. Plugins that you install manually

  2. Plugins which get installed automatically by third-party software (For example, various annoying toolbars)

Earlier, these third-party plugins will be installed without your knowledge. But now, Firefox want you to manually approve them before installing. If you already have them on your old browser, it will ask you to choose only the required plugins while upgrading :

Firefox 8 also comes with a very crucial feature which will allow us to speed-up browser startups. If you are letting the browser to remember the previously opened tabs and load them the next time the browser starts, the browser have to load all the pages which will take a huge time for it to load. But now, with a simple tick, you can set it to load the pages only if you select the tab. The frist tab will be automatically loaded but the other tabs will be blank until you switch to them. A really great feature!

Apart from those features, there are a lot of minor bug fixes and security patched on this new version. So, in few hours Firefox will notify you to update your browser. If not,

  • Go to the Firefox about page (In the menu, help > about) and it will check for updates

  • Or, download the new version from the official site and install it!

For Linux people, we’ll check whether they have updated it on the repos and if not, we’ll try to cook up a solution. Currently the Mozilla PPA is down. Lets hope they will work soon!
