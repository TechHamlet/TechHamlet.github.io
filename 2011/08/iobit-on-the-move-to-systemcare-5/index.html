IObit on the Move to SystemCare 5

Aug 24th, 2011 | Comments

After the big rush of SystemCare 4 version which changed the way we looked at system caring now IObit is going to release SystemCare 5 sooner than ever. Currently the Version 5 public beta is out and you can download it at their forum. But still it’s not ready and isn’t recommended to be installed in normal pcs. So if you like to check it out just get yourself a copy of a Virtual Machine and try it out.

In the forum post there are few New updates that are mentioned.

  • A new architecture is introduced and source code is rewritten to 32bit and 64bit systems.

  • Real time optimization for ActiveBoost function

  • New cloud technology

  • Improved UI so that system resources used is minimal

and much more. You can look at the orginal post to check out some more features and after downloading it you’ll be surprised by more.

Another thing we have to mention is that IObit release cycles have increased. The time take to roll out Version 4 from Version 3 is really big compared to the Version 5 from Version 4. So you’ll definitely see more improvements. Maybe IObit is following Mozilla’s lead towards faster updates and faster versioning to improve the project. What do you think?

Update : The final version of Advanced System Care 5 is here!
