Ubuntu Customization Guide : The U Bot!

Aug 11th, 2011 | Comments

Unlike Windows, you can customize Linux in anyway you want. There are thousands of themes, icons and software all around the web to help you. Not only that, you can even build your own theme if you know what you are doing! No one can stop you! But, the problem is, most of the new comers, sometimes think that the Linux look sucks… So, they leave without even saying bye! That’s why we started to prepare awesome customization guides which every beginner can use to light up their desktops! Actually, this is the 3rd guide in the series!

Both of our earlier guides (A Night at The Hamlet & Purple Boy) used simple easy to install themes with some screenlets and some small additional apps. This time, things are going to get a bit harder… Earlier, we never included Conky in our guides. Thesedays, its one of the main decorations which sits on many Linux desktops! Normally, configuring Conky is a bit hard. But, with the latest release of Conky Colors, everything became much more easier! Enough talking… Lets start the work :

First, lets install Conky colors :

  • To install Conky, you better follow the official guide (Its well written and anyone can understand it! So, I don’t have to re write it.). You can find it on this page : http://helmuthdu.deviantart.com/art/CONKY-Colors-244793180

  • After you install Conky Colors, remember to install and configure the conky weather forecast script (If you don’t install it, the new Conky styles won’t work!) : http://ubuntuforums.org/showthread.php?t=869328

  • After you have completed everything, run the following two commands to run conky (Edit the w(width) and the h(height) values with your own screen resolution) :

  • Note : If you want to start Conky with Ubuntu, please check the official guide

Done with Conky? Actually, you might see a small difference in my above instructions. Less instructions! If you stayed with our blog from the begging and followed the previous guides, you might already know how to go around Linux! So, now, things are going to get tougher!

After you are done with Conky, Lets install the clock screenlet :

  • To install Screenlets, open your terminal and enter the following command :

  • After you are done, open the Unity menu (Click the top left corner) and enter “Screenlets” to find the app

  • Inside Screenlets, double-click on the screenlet named “Clock”

  • Now, you’ll see the clock ticking in your desktop!

  • Right click on it and click on “Properties”. Now go to the Themes tab and click on the theme named “Sensor” (Note : You are free to install any theme you want)

Now, Lets Set up the Theme and the Icons :

  • The theme I used on this desktop is this : http://gnome-look.org/content/show.php/Divergence+IV+–+%22A+New+Hope%22?content=133892

  • Go to that link and download the theme zip file and extract it

  • Now, right click on your desktop and select “Change Desktop Background”. Go to the “Theme” tab and install the tar.gz file located in the extracted zip file

  • The same procedure applies when installing the icons

  • Download the Icons from here : http://code.google.com/p/faenza-icon-theme/downloads/list

  • In here, you don’t have to unzip the file because its already in the tar.gz format. Just open “Appearance preferences” like before and install the icons. After that, go to the Themes tab, click on “Customize” and go to the “Icons” tab. Then, choose the icons you have just installed. (Note : I have selected “Faenza-Darker”)

Finally, lets add the Ubuntu Bot!

  • First of all, lets say “Thanks” to HowToGeek.com for finding this wallpaper. Actually, they have a really good set of Ubuntu wallpapers in that page. You better check them out. The one I have used is this : http://technology.desktopnexus.com/wallpaper/625334/

  • Download it and apply it by going to the “Appearance Preferences” window > Background

Now, lean back and have a look at your desktop… Like the new look? As I said earlier, this time, I reduced the number of instructions. So, if you get stuck while following this guide, please contact us and we’ll be glad to help you! If you managed to do it by yourself, don’t forget to leave a comment and let us know what you think about your new desktop!
