[Ubuntu + VirtualBox] How to Share a Folder With the Virtual Machine

May 2nd, 2011 | Comments

I started to use Ubuntu with VirtualBox to get ready for my Wrestling match with Ubuntu and jBoss. Now I wanted to share a folder from my PC to the Virtual Machine so that I can transfer jBoss to Ubuntu. I couldn’t find any detailed enough on how to do it so I mash up the courage to find it on my own. So I got the doc of VirtualBox but it didn’t help much.

So I end up writing this post about it. Do mind that this is Ubuntu 10.10 and VirtualBox 4.0.6. So first things first.
