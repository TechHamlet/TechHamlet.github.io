How to Setup Your Own Mini Broadband Wireless Network

May 2nd, 2011 | Comments

Broadband routers are the easiest way for you to build your own network, wired or wireless internet can be built using a broadband internet. All configurations to the network depend on the owner of the network. There are several methods of configuring your network, but the most important thing you need to take note of is your password.

What can a Broadband router do?

Broadband router automatically shares your broadband Internet connection to all computers connected to the network. You also can configure it to limit Internet access based on several criteria (for example, time of the day that you may want your employees to be able to access the Internet only during lunch time or after job).Hardware firewall also Prevents several kinds of attacks to your computers and also prevents shared folders and printers on your local network from being assessed by computers located outside your home or office. Most broadband routers usually come with a 2 or 4-port switch, allowing you to connect the computers you want to connect to the network directly on the router. You can also expand the number of ports by installing an external switch to the router. So if you are running a small network with up to four computers only you wonâ€™t need any extra switch to build your network.

The most recent broadband routers have wireless option that can allow you to connect computers without using cables. However, they will need a wireless network card installed on each computer on your network to connect to, this can be expensive. But this is the best solution for your home or small office where you have one or two laptops with wireless capability. All you need to do is to turn them on and you will be online. However, there are a lot of security risks and fancy configuration options for using the wireless option.

What is needed to setup a broadband wireless Internet:

Of course at first you will need a Wireless router, a networking cable made from a cat5 twist, a broadband portable or non-portable internet modem, a system with a NIC interface and then you are good to go.

How can you set it up :

After the installation of the broadband internet modem which you have done on your system, next is setting up your wireless router to share your internet through your network, then next is configuring your wireless router, just follow the step by step guide to configure it.

The security authentications that are supported are ; WEP, WPA/WPA2 Â with a pre-shared key or radius servers. Where to start your configuration, First, Open your web browser (preferable firefox or internet explorer) and then type 192.168.1.1 or 10.0.0.1 or the default IP address of your wireless router and then hit the enter button.

After the page appears, you should be seeing a pop up page prompting you to enter your username and password, for this check the manual packed with your router when you purchase it. After you might have done this you will see the router welcome page where you will set a lot of features like WAN, LAN, DHCP, SSID and other basic settings that will enable you to use your router successfully.

WAN: Wide Area Network settings:- Can be setup right on the first tab area where you will set the IP address that you network provider has given you to access the internet. It can be any of these options

  • Automatic Configuration DHCP: where the IP address is obtained automatically from you network provider.

  • Static Configuration: where you will be required to type all the network parameters given by you network provider.

  • PPPoE Configuration: this option can only be set if your network provider uses PPPoE, here youâ€™ll be provided with the username and password, and the settings are only used for DSL cable services.

And so on, these settings must be followed strictly to avoid errors.

LAN: Local Area Network settings at the same interface you will need to set up the router IP address and gateway, For example since your Home or office network is private it should be set as 192.168.1.1 and the subnet mast :255.255.255.0

DHCP: Dynamic Host Protocol settings. A lot of router have in-built DHCP server, the server will automatically assign an IP for all computers on your wireless network, and on the same setup page you must set a starting IP e.g. 192.168.1.2.

SSID: Under these setup you can set the wireless network name (SSID) to be easily identified by all computers connected through the wireless network, SSID settings are case sensitive and shouldnâ€™t be more than a 32 alphanumeric characters, still on these same setup page you can set your Wireless channel, Wireless SSID broadcast and so on.

More under these settings is the security setting where you set your own choice of security, like WPA and so on. After all these settings are already properly inserted then you are good to go browse your broadband wireless internet.
