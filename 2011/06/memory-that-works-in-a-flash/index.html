Memory That Works in a Flash

Jun 10th, 2011 | Comments

A couple of weeks ago, Sony unveiled the fastest ever memory stick – the PRO-HG Duo HX. There are many companies that manufacture memory sticks today, but it was Sony who introduced the memory stick back in October 1998 and was available in memory sizes up to 128 MB.

A memory stick uses flash memory which is a type of electronic memory which offers easy and fast information storage in computers. Flash memory is classed as a solid state storage device and this means it has no moving parts. Flash memory came about as there was a need for devices like digital cameras and mp3 players to have a small portable memory which was large enough to store all your photos or music. Using a magnetic storage system such as a hard drive would obviously be too large, and because it uses moving parts to function, would not be suitable for carrying around with you. Therefore the flash memory was introduced which was perfect for small portable electronic devices.

Computers store information as binary code inside their memory chips. A memory chip is an integrated circuit made of millions of transistors which can either be on or off. On corresponds to one in binary and off means zero. Transistors are simply electronic switches that can be turned on or off by electricity so in effect, an electrical state is being used to store data. However, the problem here is that when the electricity is switched off, such as when a computer is powered down, all the information is lost. Memory which is lost when the power is off is called Random Access Memory (RAM).

Flash memory gets round the problem of memory loss after power down by using a completely different type of transistor which keeps its  switched on or off state even after the power is off. Flash memory is a type of Electrically Erasable Programmable Read-Only Memory (EEPROM). An EEPROM chip has a couple of major advantages over a ROM (read only memory) in that the entire chip does not have to be completely erased to change a specific portion of it and the chip does not have to be removed to be rewritten.  This is why a usb memory stick is so versatile for carrying data as information can be written to it, read from it and erased from it with ease.

The future of Flash memory looks bright as this technology is becoming more and more widely used. The number of Solid Sate Drives (SSD) to be shipped in 2011 is estimated to hit 55 million. SSD’s incorporate Flash technology and are being used more and more inside computing devices like the ultra slim MacBook Air and MacBook Pro. In addition to this, the latest Windows operating system, Windows 7 is optimised for SSD.
