How to Set Up FTP in Dreamweaver

Jun 11th, 2011 | Comments

Setting up FTP (File Transfer Protocol) for Dreamweaver will make it easier for you to update your website. For those of you who don’t know what FTP is, it’s basically, a way of copying files from one host to another (uploading files from a computer to your web server). There are many free softwares that let you accomplish this work. One of the most commonly used FTP softwares nowadays is FileZilla – free and available to download! However, lets look at, how to use FTP on Dreamweaver itself.

Follow the steps below:

  • Open Dreamweaver, on the menu bar - click Site, New Site…

  • What would you like to name your site? I suggest you to use the name of your website. Eg:– TechHamlet.com

  • What is the HTTP Address (URL) of your site? Eg:– http://www.techhamlet.com

  • Click Next >, select No I do not want to use a server technology (unless you want to work with a server technology)

  • Click Next >, select Edit local copies on my machine, then upload to server when ready

  • Where on your computer do you want to store your files? Select a directory that’s convenient for you!

  • How do you connect to your remote server? Select FTP from the list

  • What is the hostname or FTP address of your Web server? Eg:– techhamlet.com

  • What folder on the server do you want to store your files in? public_html/

  • Input your FTP login details (username and password)

  • Check (tick) Save and then Test Connection

  • A message box will pop-up saying Dreamweaver connected to your Web server successfully

  • Click Next >, (I suggest you to) select No, do not enable check in and check out – to make sure, only one person can edit a particular file at a time

  • Click Next >, Done

Different people have different web hosting providers. Your provider should give you the information you need, to set up an FTP connection. Please contact your Web hosting provider if you find difficulties in setting up the FTP connection.
