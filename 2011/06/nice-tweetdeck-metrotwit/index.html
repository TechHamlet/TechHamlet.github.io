Nice TweetDeck - MetroTwit

Jun 19th, 2011 | Comments

You all know about the sale of TweetDeck right? It’s the kick-butt application to manage twitter without any hassle. But one thing everyone will say is that it’s not the nicest app out there. Compared to TweetDeck, Sessemic is lot more beautiful and simplicity is integrated gracefully. Plus you must have heard about the Microsoft’s Metro UI right? It is one of the features that going to make Windows 8 a big hit since the interfaces gets more glamor with Metro UI. Now there is a Twitter app out there called MetroTwit that you should definitely check out.

Jobs bragged for years about the beauty of Mac apps and the Apple’s design principles. He was the person who included beautiful typography to the Computer world. But yet I got to agree with Microsoft with this one – Metro UI might be the magic that can put Apple in havoc considering design. MetroTwit App is built on .NET Framework 4.0 and it’s specifically built for Windows. Considering TweetDeck that supports over multiple platforms this App utilizes the special benefits of .NET framework.

Simple layout makes the App easy to use and more fun to interact with. Plus it’s like App in the Cloud(real clouds). Even the beautiful web interfaces we love can be surpassed with Metro UI since it definitely has a promising future.

Most common features of TweetDeck is included in MetroTwit so no point summarizing them. But I am going to point out few interesting features of MetroTwit.

  • Windows 7 taskbar integration - A count will show how many unread tweets there are in a bar

  • Themes - Customize the user-interface the way you want with colors. It’s the best way to make the contrast you want.

So what’s the big deal about MetroTwit? Compared to TweetDeck still MetroTwit has a long way to go. MetroTwit’s greatness revolves around Interfaces and not functionality. But this is a quick snapshot of future Windows interfaces. Apple better prepare itself because Windows might beat Apple in their own game of interfaces.
