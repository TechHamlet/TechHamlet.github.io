Go Daddy vs. Network Solutions

Feb 16th, 2011 | Comments

[ad#Dulitha Wijewantha - Lead-in AdSense Text]

As the title suggests I wanted to review Go Daddy and Network Solutions for a long time. The heat between the two companies have grown up so high that Network Solutions have produced a parody video about Go Daddy called as Go Granny. The direct problem we find with both of these sites is that many users have complained about them both.

In one forum one user has mentioned that Network Solutions is not Customer satisfactory but in another Go Daddy gets the same complain -

The below complain is about Network Solutions -

I’ve only heard negative things about Network Solutions to be honest. The first time I heard about them was when I first joined Namepros. I didn’t take the negative reviews seriously…until one of my friends tried registering for a domain name. He used a site to check the domain name’s availability…5 minutes later he was about to register for the domain name and all of a sudden it was taken! So he asked me to help him out because he really wanted the domain name. What did I find out? Network Solutions registered the domain name (i.e. it was “available” on Network Solutions but other sites incl. GoDaddy said it was already taken). I know it’s nothing personal just business but seriously…that is sick, period. I told my friend not to check again, until the grace period was over. Fortunately, he got the name he wanted after the wait.I think you should register at GoDaddy, it’s well reputed and the domains there are cheaper than Network Solutions. (Source – youngentrepreneur.com)

[ad#Dulitha Wijewantha - MidPostAd]

The below complaint is about Go Daddy -

October 2009. NS has been updated the prices and packages schema. Some host packages are expensive than GoDaddy, but the panel control of NS services is better, as I see. GD is slow to enter to the host panel. Is too difficult (impossible for me) to redirect the domain root to my Jommla instance directory. Finally end installing the instance on the root host !! NS works fine this issue. My site under GD was blocked by Fortinet engine because the GD Ip was relations with adult content sites. By writing to Fortinet was possible unblocked my site. With NS there are some problem with manage group policies to all email accounts hosted on there. If I need to block some email address to Incoming, for example, should go each account doing filter (are >100 accounts) There is no way to unblocked a email precalif like spam by Symantec Brighmail engine. At the least, is not perfect, is expensive, but I prefer NS. (Source - itedge.net)

But according to the prices the Go Daddy seems the best solution because the prices are Comparatively cheaper. But on the bright side for Network Solutions they look more professional than Go Daddy. But in another review site you can find out that Go Daddy lacks customer support and their response time is more than the Network Solutions. Go Daddy is considered as the world’s largest domain registrar and Network Solutions is also well reputed. But we currently at no position to analyze both companies because customers are giving different feedbacks about the services offered.
