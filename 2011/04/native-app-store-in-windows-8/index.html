An App Store in Windows 8?

Apr 12th, 2011 | Comments

Windows 8, the upcoming member of the windows family is said to have an inbuilt App Store. App Store is a place where you can get Applications for your platforms easily. Screenshots have been released about this App Store but it appears that their will be slight changes in the original release.

But then again I still have a problem with the need of it. Because App Stores are necessary when Software are hard to find. Come on. Windows products are very easy to find. App Stores are essential when software are hard to find for a particular platform. Then again there is a problem regarding Amazon using the word App Store with Mac. One positive point I can think of is the hope that this App Store might bring Cheaper software to the market because single developer Apps will have a chance in App stores.

The design too should be spiced up and made into shape and add some features other App Stores doesn’t have.

As you can see, even the MS Office package will be available at the App store.

There seem to be a gifting option in the App store too.

What do you think about the Windows 8 App Store that is about to be released? Will it help us out?
