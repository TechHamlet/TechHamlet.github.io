Code From Anywhere, Anytime

Apr 1st, 2011 | Comments

That’s right. Now you don’t have to take your files nor your favorite IDE when you want to code some where else rather than your workspace. Cloud9 an amazing service featuring the Ultimate Javascript IDE will help you. Cloud9 has the state of the art tools to enable you to get serious coding. Plus they even allow free account for Open Source projects and 30 days trial for commercial projects. More than that Cloud9 has instant access to GitHub, the popular Code repository.

Cloud9 has many advantages. Number one being there are relatively less number of IDEs for Javascript. Even though Dreamweaver has support for Javascript files, any Javascript developer knows that it has very poor support for Javascript debugging and coding.

  1. Cloud9 lives on the Cloud- Your code is on the cloud and that means you only need a modern browser to view your code and edit it on the go

  2. Awesome Code Editor - The code editor is good and shows errors that are very subtle. Like the no ending bracket typo. **

But I couldn’t find auto-complete on Cloud9, that is one downside I saw on Cloud9. And another thing is that for CSS editing on Cloud9 errors are not show.

**

Testing CSS

You can see that the universal (*) style block didn’t end with a bracket and this wasn’t show on the Cloud9 notifications and warning like the before Javascript error example. And this holds true for xml documents too.

**

I haven’t tried out python, Ruby and PHP(I don’t know them~hehee). I hope that Cloud9 adds jsp and servlet support too their IDE.(I am coding in Java)

  1. Collaborate simultaneously - This is an excellent feature. When working on large projects developers have to work together and sometimes developers change and many things happen in development phrase. With Cloud9 you can edit the permission of developers easily and remove or add developers seamlessly.

  2. Chat on the IDE - Yeah… You don’t need not to have Skype or AIM running on the background to act ad the communication tower. IM clients reduce the productivity because it’s not embedded directly on the IDE itself. But Cloud9 has transcended this barrier and have taken the brave step to add the IM client directly on to the Cloud9 IDE.

  3. Preview is easy - One click button is there to preview document we make and this is great. Error! Filename not specified. But to preview server-side pages you will to configure a server. And I tried to do this. But I think I found a bug on the platform. Here is what you get when you press the configure server button. **

**** But for some reason a browse box didn’t open when I press the browse button. I don’t know whether this a bug. I reloaded several times but didn’t work.

  1. For Open source it’s free - If you are a fellow developer who is working on an open source project. Then Cloud9 is the right solution. Because it free for qualified Open-Source Project. Even for Commercial Projects it’s 0.49$ per day.

Cloud9 is a great feature for Javascript development and I really hope that they Cloud9 developers would add jsp and Servlet support to it. Signup today for Cloud9 and see how it works. You might like it very much. Plus you get a 30 days trial too. I am very curious about IDEs too. Another alternative I found for Cloud9 is Aptana. Well… Aptana is good. But Cloud9 lives on the cloud and helps collaboration. That’s the awesome part of it.
