Lets Dive Into the New Ubuntu 11.04 [Just Released]

Apr 28th, 2011 | Comments

Just now, the Canonical team gave birth to a small toothed whale called Natty Narwhal (Ubuntu 11.04). This latest addition to the Ubuntu family is going to offer a huge change to the whole OS. Ever since Ubuntu released the plans of Ubuntu 11.04, it became the hottest topic in the Linux world. The biggest reason is Unity! Want to know why? Want to know whats new? Then prepare your scuba gear because we are going to dive!

Earlier, Ubuntu came with several editions. Desktop Edition, Netbook Edition and the Server Edition. This time, they have stripped out the word “Edition” from all the flavors. Not only that… Remember the difference between the Netbook edition and the Desktop edition? The Netbook Edition of Ubuntu had a special shell called Unity to power the desktop. Unity was built by Ubuntu as a replacement for the Gnome shell. Their own desktop on their own OS! Pretty cool isn’t it? Actually, this made Ubuntu Netbook Edition very popular. But, Ubuntu announced that they are going to bring Unity to the Desktop Edition! A huge crowd like this idea, and just like that, another huge crowd hated it. Unity had a lot of stability issues in the beat versions. But remember, beta versions always have issues. Some even didn’t like how it worked, while another set of people liked the change in Unity. So, Unity became the hottest topic in the Linux planet! Recently, we held a vote to check how many people liked Unity. Unfortunately, most of our readers hate it! You can check the results from here : Ubuntu : What do You Think About Unity? [Poll Results]. Now, there’s no difference between the Netbook version and the Desktop version. So, both of them are merged and now you’ll get only Ubuntu Desktop and Ubuntu Server.

When speaking about major software Updates, Ubuntu 11.04 comes with the latest Firefox 4, Banshee 2 and LibreOffice (Which will replace Open Office). The new system is built on the latest (stable) Linux kernel 2.6.38.

The new Ubuntu software center also have some great changes. Now, the users can review and rate every software in the software center. So, you can look at other’s experience on that software before installing it! Earlier, Ubuntu software center was slow at the start. Now, it have improved its loading time. You can also buy premium software right through the software center.

When it comes to looks, Ubuntu 11.04 have a completely different look and feel due to Unity. It also comes with a brand new set of Wallpapers. Actually I didn’t like the wallpapers In Ubuntu 10.10. But the new version have some attractive photos. Also it have several small changes like a brand new scroll bar (Which you haven’t seen in anywhere else). But, be careful, when activating some compiz features. Theres a known issue with Ubuntu 11.04 and the compiz Desktop Cube.

Still Natty have some bugs which you can find in their site. One of the biggest bugs I faced is the failure to detect my Nvidia drivers. If I set the Ubuntu installer to download and install the latest updates during the installation, my system won’t boot! So, all Nvidia users, better watch your step!

That’s not all, there are more cool features in Natty Narwhal. To learn about them, head over to Ubuntu’s home page and grab a copy for yourself! Don’t forget to tell us what you think about Ubuntu 11.04 as comments!
