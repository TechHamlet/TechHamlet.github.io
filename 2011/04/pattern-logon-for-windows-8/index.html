Pattern Logon for Windows 8

Apr 24th, 2011 | Comments

More and more Windows 8 news are leaking out now. We have found out that there is going to be an App store in Windows 8 and there will be a different UI style again including the Ribbon feature for explorer too. From the latest reports we can say that Windows 8 will feature a pattern Logon like Android.

Even a video has been leaked that shows the Windows 8 pattern locking login screen. You can view this video on Youtube but do mind that the Video is bit NSFW. More than that we are revealed to the Metro UI interface used in Zune and Windows 7 mobile. We can expect a cleaner and faster UI in Windows 8. Comparing the history we can see that Microsoft always does improve their interface system. Look at Windows XP and Windows Vista. Then take a look at Windows Vista and Windows 7. It’s just mind-blowing UI advancement.

Another intense situation is also sticking with Crackers bypassing the Windows 8 M1 Builds. They are using Windows 7 files to remove the non-Genuine flag and the Time Bomb on the Builds. What’s more is that Crackers are getting their hands on the hidden features of the OS. More than that we can expect to see Windows 8 build copies in the internet soon because the Builds can also be made to ISO images easily.
