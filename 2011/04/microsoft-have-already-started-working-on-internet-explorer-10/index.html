Microsoft Have Already Started Working on Internet Explorer 10!

Apr 14th, 2011 | Comments

Last month, Microsoft released a major update to their Internet Explorer. Usually, they wait for about an year before starting the next version. But, this time, things are going to change. Most of the web browsers like Chrome, Opera and Firefox all have accelerated their release cycles to stay up to date with the latest technology. Now, Microsoft have also decided to go with the trend!

“You can run these at www.ietestdrive.com to see emerging standards like CSS3 Multi-column Layout, CSS3 Grid Layout and CSS3 Flexible Box Layout, CSS3 Gradients, and ES5 Strict Mode in action.”

“IE10 builds on full hardware acceleration and continues our focus on site-ready Web-standards,”

-Dean Hachamovitch (IE corporate vice president)

What’s so special about IE 10? For now, IE 10 don’t have any special feature additions like we had in its previous version. According to Microsoft, they are just going to improve CSS3 and HTML5 support and work more on the hardware acceleration features. So, it will be some good news for web developers.

Yesterday they released the first platform preview of Internet Explorer 10. So, the fun begins! But remember, platform previews are only for the developers and not for the normal users!

Actually, with the release of IE 9, Microsoft hoped to become the champion. They got a huge amount of downloads, but people said that Firefox have a bigger record than that. Just after few weeks of IE 9’s release, Mozilla released Firefox 4 which managed to get about 7 million downloads in the first 24 hours beating IE 9 and its own previous record! Now that’s a huge blow for Microsoft. I think that’s one of the main reasons which have forced them to go for IE 10 as soon as possible. But, I say they’ll need a miracle to get a good name in the browser market again!
