10 Days Course to Master CSS

Apr 21st, 2011 | Comments

When we are talking about CSS and Web design, they both go together. I am compiling 10 tutorials that were posted that you must do to master CSS easily. Since CSS3 is also out now and advance functionalities can be added to our designs easily. If you are new to CSS and Web Design these tutorials will help you to be a pro in the trade with small amount of time. I am including reference links to the tutorials and further tips to help you get the most of the tutorials. Most important thing is to do one tutorial a day and experiment more about it on the same day.

Day 1

Ok.. now lets get started. I first found out this tutorial to be very helpful. This actually doesn’t have a tutorial in it. But consists of 30 CSS selectors that you should Memorize in order to master the trade. Try using these selectors and doing something with them until you can actually remember all these selectors and there use. Then you are ready to master what to use at the give spot. Word of caution the nth-child selector doesn’t work in IE 8. Test in all four major browsers(Firefox, Chrome, Internet Explorer, Opera).

Day 2

Buttons are one of the most important parts of Design. Layout is another one too. To start of on the second day lets get the buttons done. Then you’ll be able to make kick-butt CSS buttons that will spice up your design. Plus play around with the code and feel free to refer things you don’t understand at the W3schools. Another tips to button making is try out this amazing button making framework. You’ll be able to make great buttons easily with the framework(I call it a framework. Feel free to call it anything you like). But remember to go over the tutorial below to really get the feeling of actually making it with CSS. Another tip is that there are many frameworks out there and you need CSS skills to debug those frameworks when you get bugs. So don’t skip the tutorial below.

Tutorial

CSS3 Buttons with Icons

Day 3

Now it’s day there and you are ready to get started with a form building process. You’ll get many places to build forms. Customer support forms, Signup forms, Contact Us forms, many more. This is the way to spice up your forms by rounding the corners of your input fields. Using shadows and making a better UI in the end. You can combine a button for the send and reset from the previous day and stay remembered about the button making too. Now use the Framework because we need to focus on the form it self. One more tips with these tutorials is not to copy and paste the code from the tutorial itself. I once did that with a tutorial when I wanted to build a content slider. I took 3 days to build my own one with the concept and to design it. It’s just two hours work if you do everything from the start understanding the code. Then all these tutorial will turn out to be very easy to implement to your actual designs.

Tutorial

Design a Prettier Web Form with CSS

Day 4

Cleanliness is a habit. Remember that! Because I always had a bad habit of not dropping comments while coding CSS. This made me sometimes completely delete style sheets and write new ones! Any code/CSS write without comments is code/CSS you never wrote in 1 months time. Even sometimes weeks. Commenting, indenting, meaningful class names are all must. Don’t worry about the file size because in the end you must drop the CSS to a minifyer to reduce the file size. You’ll find bugs with minifying too. So learn CSS more and more in order to debug bugs you might meet in the future(Specially CSS3 related bugs).

Tutorial

5 Time Saving CSSEdit Tips

Day 5

You will come across many times when you want to give a message to your visitors/users easily. Like a quick alert. You can use Javascript to do something like that. But this tutorial will teach you how to do it with CSS. You can use jQuery to do it and I once did it with CSS too before I found this tutorial. Well my one lacked animation and the wizzz around the message. But you must consider about browser compatibility when using techniques like these. Always remember to make sure you develop anything with progressive development techniques that will not leave some of your visitors not seeing the site.

Tutorial

CSS Modal

Day 6

When using CSS you’ll probably note the fact that the browser will inflict your design with default CSS styles. These styles are always getting the way because sometimes they add padding, margin, border to some elements. I usually use

*{border:0px; margin:0px; padding:0px;}

in my stylesheet when I am first starting. But when it gets to form designing and typography you’ll need to use a special reset stylesheet to get away with these problems. This tutorial will help you to learn them. My approach is best when starting but when things get serious always go and change these styles to the core. Particularly the input elements styles.

Tutorial

Default Styles for CSS

Day 7

Now is the time to learn more about floats and to make advance layouts. Layout is very important because in the end you are making Web Sites. Good web sites have good design with good usability. While making layouts you’ll probably come across floats, clear, positioning. Learn these very well. They are like the core to a good CSS designer. I am using 2 tutorials here to let you understand it better. I am repeating again that you must experiment with these new skills otherwise you’ll never learn the maximum power of CSS.

Tutorial

All About Floats

Day 8

Now is the time to automate things to help you out. Because there are lot of CSS generators out there. But remember this, generators are mostly useful for CSS3 properties like box-shadow, text-shadow, gradients. So don’t look so much for layout generators. I am not recommending any because I have used many before and found out that it’s always better to this part ourself. Use firebug to understand the widths and the heights in the webpage and make a good layout without generators. Remember to playaround with :hover selector too to add cool responses to users activities on the page. These responses makes the website interactive and responsive.

Tutorial

What CSS3 Can Do For You: CSS3 Generators and Other Helpful Tools

Day 9

Now comes the Menu building Day. Get ready to build some amazing menus. Menu is one of the most important part of any website. Because the menu clearly points users to find information they want. Do the below tutorials and resort to make your own menus. Trust me it’ll be a piece of cake when you get the hang of it. Sometimes it’s not the best practice to have a drop-down menu. Well…. Drop-down is cool and everything but remember simplicity and minimalism is something every designer should keep track of. One site I want you to have a look at is UserVoice . Look at there menu. You’ll get a good understanding of menu designing from them. First ask the question whether you want a Drop-down menu or not. If you answer yes then ask yourself do you whether you have enough navigational hierarchy. If not then chose a normal menu.

Then find out more about the points and you’ll get a better insight about web design. You should probably be through with CSS now. But we all have to learn. So signup for updates from good CSS blogs.
