Ubuntu Tweak 0.6 Beta Hits the Streets!

Sep 26th, 2011 | Comments

Many Ubuntu users used Ubuntu Tweak as the control center of their system. A small app designed to tweak and maintain your whole Ubuntu system… Actually, many people loved this tool because it allowed you to do many things without diving deep into the system. In November 2009, Ubuntu tweak blog announced a brand new Ubuntu Tweak 0.6! Well, we just received its first beta!

The UT team have been preparing for this new version since 2009! When they first released their design ideas, many people liked it. A lot of positive comments showed many people are looking forward for it. So, we started counting and on April 2011, again they showed some screenshots of the new version. But still we can’t test it!

Last week, another interesting announcement came from Ubuntu Tweak… This time, they have a working beta for us! Cool! Actually, they didn’t release any major versions for a long time. As I said earlier, they started drawing plans for this since 2009. So, they have plenty of time to design a great tool.

Now, we have a nice home screen which shows a summary of your system. Your hardware, Ubuntu Information, Whether your software are up to date, Whether there are any junk in your system and your User information. It also shows a list of recently used tools on the right side of the screen. Mainly, the system is divided under 4 main tabs which groups all the tools. As usual, most of the system configurations which takes some diving in can be done very easily through these tools. A minimal and a well group designed!

Now, there’s something new. Earlier, we had a feature called Package Cleaner in Ubuntu Tweak 0.5. In 0.6, they have re-named this tool as “Janitor” and have added more features which can scan and clean your system. If they can add some comprehensive cleaning like Bleachbit, that would be great!

That’s what we can see from the outside… Now, let’s go to the inside! Ubuntu Tweak 0.6 is not all about the looks. There’s something big going from the inside. The main target of this new system is to let others to make the system better. Whats that?  That means, this new version have a plugin system. Now anyone can build plugins for UT and make it more stronger. Add more features to it… Cool isn’t it?

Omg, the Application Center is missing! One of the most popular features in UT is the Application Center which allowed us to install a lot of apps that doesn’t come with the system by default. It also had a Source Center which helped you to add sources of many useful apps. Don’t worry, they are not stupid to lose such a cool feature. The developers are currently building that feature as an extension. But it’s not yet ready. Don’t worry, you’ll be able to try it soon.
