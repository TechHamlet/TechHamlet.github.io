Ubuntu 11.10 Beta 1 : An Early Tour of Oneiric Ocelot

Sep 2nd, 2011 | Comments

On April 2011, Ubuntu released one of their most controversial releases Ubuntu 11.04. But, it looks like the team is very confident with their change and they are going to continue what they have started. In the next release of Ubuntu, Unity is going to be more stronger and the whole system is about to get a lot of new updates and features which might be able to change your mind!

Because of the new Unity shell Ubuntu introduced in their previous release, many people started to shout against the developers. Not only Unity, even Gnome 3 have the same issue (Even the father of Linux hates it!)! Actually, the main reason for this problem might be the difficulty to adjust into new environments! So, its hard to blame the developers… For now, lets forget about the big problem…

The first thing you’ll notice after installing Ubuntu 11.10 is the brand new login screen. Now they are using Light DM which is a lightweight login screen which can be dressed up easily. Actually, we covered it in one of our early posts : Ubuntu 11.10 Will Implement a Better Login Screen with LightDM

In Ubuntu 11.04, the developers worked hard to integrate Unity into the system so it didn’t get much new features other than Unity. They didn’t even update Gnome. But, the next version will get Gnome 3.1 with the latest Linux kernel 3.0! Actually, whats special about Gnome 3 is its new UI. But, Ubuntu users already got a similar experience through Unity. The new Linux Kernel also have a lot of good news for people who have new systems. It contains many drivers and inbuilt support for new hardware like Microsoft Kinect, AMD Fusion, Intel Ivy and Sandy Bridge processors and many new graphic drivers. According to Linus, there’s no scary new features in the new kernel : “No breakage, no special scary new features, nothing at all like that”

There’s another big change… First of all you’ll notice the new icon which represents the software center. Not only the icon… The whole Software center have a brand new look. Actually, I was unable to take a screenshot of it because it doesn’t load correctly in my test machine. But, check our previous post which covered the news about this new change : We Saw a Brand New Ubuntu Software Center in Oneiric Ocelot!

There are also some new improvements to Unity. The new release will make Unity more stable by eliminating a lot of bugs. The Unity dash also have a new look. The icon have changed and you’ll also see a new set of icons and ratings! It also shows apps which are available for download… Hmm… There are much more to discover in this new Unity…

New interface changes, Light DM, Software Center, Gnome and the Linux Kernel… Those are the big changes coming in Oneiric Ocelot. Apart from that, there are a lot of small changes added to this new release. Actually, my personal thought is that, things are getting a bit more stable in the next release… At first, I hated Unity but now I think its time to get used to it! What do you think? Not only that… There’s another big reason for me to count my fingers untill they release Ubuntu 11.10. The current version of Ubuntu doesn’t work well with some new laptops which have switchable graphics. I have the same problem… All I get is an evil black screen! Its more evil than the Blue Screen of Death in Windows because the BSoD have some life (text). but this black screen is completely blank! Hope the new kernel will fix these issues…

If you want to try out the new beta version, install it in a test machine or in a virtual environment (Use VirtualBox to create one). But, do not install and use it in your own machine… Because, beta means it have bugs! To tell you the truth, when installing, even the installer crashed in my test machine! So, better watch yourself!
