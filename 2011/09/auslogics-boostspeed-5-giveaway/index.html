Update : Auslogics BoostSpeed 5 Giveaway (5 Keys | Worth Over $249)

Sep 14th, 2011 | Comments

As I always say, if you want to get the best of your machine, you need to take care of it! It’s not an easy task… Can you manually go though your system registry and remove the bad ones? Do you even know what is the registry? Can you go through all the files in your system and remove the junk files? Can you manually go through your hard disks sector by sector and re arrange the data to prevent fragmentation? I don’t think so…

That’s why we need special tools to do this job! So, today we are going to see about Auslogics BoostSpeed! A very useful tool for your machine. Earlier also one of our authors wrote about this cool app : Auslogics BoostSpeed 5 … So, today I’m going to go through only the special features which interests me!

As soon as you open this tool, you’ll recognize the “Scan Now” button which scans you whole system for various errors and fix them in several clicks. Actually, this is something every system utility tool have… So, lets skip it…

On the left of the screen, you’ll notice some links to “Quick Tasks”. Yes, many tools have similar links with different names like “Quick Care” or “Quick Scan”. But, I see something special in here… When you hover over those links, you’ll get a tooltip which contains the keyboard shortcut for that tool. Lets say you are working on your machine and you just remembered to do some quick cleaning or free some memory for your heavy apps, you can now execute those commands by just pressing the shortcuts and the BoostSpeed will do everything in the background! Now that’s really a great feature! Here are the shortcuts which you can use :

  • Ctrl + Q + B : Erase Browser History

  • Ctrl + Q + W : Erase Windows History

  • Ctrl + Q + T : Erase Temporary Files

  • Ctrl + Q + M : Optimize Memory

Hmmm… Here’s the part which I like most… The “System Advisor”. This Advisor will scan your whole system and gives you some suggestions to optimize and secure your system. There are some tools which scan and optimize your system like this… But there’s something special in here. Unlike most of the others, BoostSpeed have described clearly what that setting does and what happens if you disables them. So, you can learn something rather-than hitting “Optimize my system” blindly!

Lets move to the “Resource Usage” tab. If you want to catch that pesky resource monster which eats up your system resources, switch to this tab. Now you’ll see the four main resources.. The CPU, Ram, Disk and the Network. Right next to the usage graphs, you’ll be able to catch the monster. In here, BoostSpeed clearly shows who’s the top user of that resource and how much does he use… A pretty useful tool for most of us…

Finally, the Advanced tools. This is where everyone stack up all the geeky tools which can be used to optimize your system madly. It’s all yours to discover.

So, my last thought is… Auslogics has done a great job on putting the most useful stuff to the front making it more easy to use. A very user-friendly app! What do you think? Want a key to try it out?
