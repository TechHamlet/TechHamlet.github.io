Taskbar Tweaker Will Add Some Cool Features to Your Windows 7 Taskbar

Oct 31st, 2011 | Comments

When it comes to Linux, many users will never let their system to live in their machines with the default settings. They customize every piece of their desktop and they have the full freedom to do so! But in Windows things are restricted. You can’t do whatever you want inside a Windows system. But, there are some small tools which makes some customizations possible and today we are going to look at a small tool which can kill your default taskbar!

7 Taskbar Tweaker is a very small app (Only 290Â Kilobytes) which can add some interesting features to your Windows 7 taskbar. First, the tool is divided into three main categories, “Taskbar items”, “Grouping and combining” and “Other taskbar options”. It’s not fancy looking like most of the tools that we usually introduce, but don’t jump into conclusions by just looking at the UI.

In the Taskbar icons section, you can customize the way the items behave. For example, you can change what happens when you middle click on a taskbar icon. By default, it will open a new instance of that app. You can set it to close the app, minimize it or to maximize it. Just like that, you can also change what happens when you right-click, change what happens when you drop an item to the taskbar, disable thumbnail preview, remove extra gap between the pinned icons and can also set it to open the pinned icons on double-click.

If you open several instances of the same app (for example, several new explorer windows) Windows will normally group them to save space in the taskbar. The second category in 7 Taskbar Tweaker controls these groupings. Apart from disabling and enabling groupings (which we can do easily without this tool), it adds several cool features like enable drag and arrange windows inside a group with the right mouse button and enable the feature to expand the groups when hovering.

Finally, the other taskbar options will allow you to change some minor settings like what happens when you spin the mouse wheel on the taskbar, what happens when you double-click on the taskbar’s empty space and what happens when you middle click on the empty space.

A very easy to use and a very minimal settings page for the windows taskbar. Personally, I’m starting to love this new tool. Earlier, I always disabled the auto combine feature in the taskbar because I need to access things quickly. I think now I can re-enable it because the 7 Taskbar Tweaker allows me to set the taskbar expand the groups when I hover over them! You will also find interesting features like this. So, try this small tool and tell us what you think!
