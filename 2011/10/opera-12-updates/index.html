Opera 12 Will Come With Better Support for the Latest Web Standards!

Oct 8th, 2011 | Comments

We already know that Opera is a great innovative browser which become popular everyday. Earlier, I wrote about how we changed Tech Hamlet with latest web standards… Well, these days, many browsers have a very good support for these standards (Except the Internet Explorer). Even the current version of Opera have a very good support for many HTML5 and CSS3 features. But now, things are going to get better!

In the latest developement release, opera have made a huge update to their rendering engine. This is the first Opera release which contains their HTML5 parser Ragnarök. HTML parsers make sure that you see the same thing in every browser. So, now, you won’t see a different site in Opera and a different site in Chrome. They have also added more support to new CSS3 features. For example, now, Opera supports radial gradients in their browser!

Yea, for normal users these things are greek… Think of it like this… Now, with the next version of Opera, your browser will render web pages better and with more nice effects!
