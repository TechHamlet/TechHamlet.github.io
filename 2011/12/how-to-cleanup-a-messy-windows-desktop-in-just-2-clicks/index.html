How to Cleanup a Messy Windows Desktop in Just 2 Clicks

Dec 7th, 2011 | Comments

First you start working on a clean computer. After working for several weeks, some of you might end up with a desktop as shown in the left image. Have a desktop like that? If my desktop start to look like that, first my head starts to spin and then, I kill! You can create separate folders and group the items on your desktop. But, if you are lazy, there’s an easier way…

I always try to keep my desktop organized so it won’t grow like that and make my head spin. I also used to delete all the unwanted stuff in the morning. But, for some of you, you need those stuff… So, if you are a geek, you might have already figured out a solution. It’s very easy… Simply :

  • Right click on your desktop

  • Click on View > Show Desktop Icons

All of your desktop icons will disappear! Still you can access them from the explorer. So, don’t worry! Please don’t blame me for writing these simple stuff, but there are people who could really use them! If you are a geek, install Linux and start following our Linux posts.
