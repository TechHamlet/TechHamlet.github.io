Dec 6th, 2011 | Comments

Every day the hackers get stronger and smarter. Viruses get more intelligent… For example, earlier, viruses went around the web blindly attacking every PC it meets. But, modern malware like Stuxnet knew exactly where to hit and how to hit! It also knew how to fool the famous security systems… So, in a world like this, we need everything we got to protect us!

If you have a full featured security suit, then you are safe (Not completely!).  But, some people like to install security components one by one selecting the best one in the category. You might have already chosen a good antivirus scanner. This is the place where many people get it wrong. Many think that the malware scanner is all the protection they need. It’s a horrible mistake!

Apart from a malware scanner, you also need a powerful firewall to protect you from network attacks. To make your system invisible to the attackers. To control your programs which tries to access the web. So, why are you waiting? Try Online Armor!

Emsisoft Online Armor is proved to be a very solid firewall system. Easy to use and configure. Very user-friendly. If you install it and go through the interface, you’ll understand what I’m telling. But if you are a power user and want more options, simple switch to the advanced mode from the home screen. I’m pretty sure that normal users can leave the default configurations because Online Armor can learn from your moves and adjust its system.

Like many other new security systems, Online Armor also have a sandbox option. If you don’t trust a software, you can run it in the sandbox (Right click and select “Open Safer”). It will prevent the software from doing any changes to your system. It will be running in an isolated environment.

Some big security systems out there does not protect you from key-loggers. We have to use separate software like the Zemana Anti Logger (But Zemana can protect you from more logging threats than simple Keylogger) to deal with these kinds of threats. But, Online Armor have a built-in Anti Key logger so you don’t have to worry about that!

Additionally, this app will monitor your hosts files which can be used by malware to redirect you to phishing sites. You might be entering the correct URL, but it might be taking you to a malicious server! It also monitors Autoruns which is the most popular method for malware to get into your machines.

Before wrapping up, need to talk about a very special feature called the Banking Mode. If you need to perform some critical business, like purchasing goods through the net, you can activate this option. It will block all the websites except for the ones in the trusted list! Now this is really important if you want to stay out of phishing sites which can silently grab your credit card details and drain them!

I used this app for several times and it worked perfectly. Very strong protection. But, there’s only one problem. The software takes some time to load and seems to be a bit slow at some places. But, still I can live with it… So, what do you think? Don’t you want a free key to try this software?
