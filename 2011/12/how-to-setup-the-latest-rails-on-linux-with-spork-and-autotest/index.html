How to Setup the Latest Rails on Linux With Spork and Autotest

Dec 8th, 2011 | Comments

Recently, I started learning Ruby on Rails and faced several problems while setting it up (Specially while setting up Sport and Autotest). After doing some Googling, I found that many people have the same problem. But, non of the solutions worked on my system. Some of them are out of date. So, I did some more research and prepared my own guide…

I’m following Ruby on Rails Tutorial written by Michael Hartl. Michael is using a Mac and have setup Spork and Autotest. Autotest will run in the background and help you to test your application in the real-time. So, you don’t have to execute rspec  every-time you do a change. Autospec will automatically do the testing and show a nice notification! If you have used Rspec, you might have noticed that it takes some time to load. Spork can make it load faster by caching some of the required files in the memory.

I’m using Ubuntu and the latest version of Rails. Unfortunately, his guide won’t work for me when setting up Spork and Autotest. I think it might be because I’m using the latest version of Rails.
