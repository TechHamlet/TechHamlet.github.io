Safety Isn't Expensive Its Priceless

Mar 12th, 2011 | Comments

The more you learn, the more secure you will be in today’s world. Day by day, the more Internet-savvy we are becoming, the more virus and malware attacks are increasing. But, if we learn how to handle the attacks and how to recover from damage made by hackers, we can enjoy our freedom in this internet world. We all know the world is running faster, no one has time to read long theories and books. So, with this in mind, here I bring some quick security tips that keep your system safe and running smoothly.

Here we go, the things you should know :

  1. Use original Windows Operating System (keep updated)

  2. Install a good anti-virus program.

  3. Update anti-virus program on a regular basis.

  4. Always use firewall.

  5. Scan your computer once a week.

  6. Create strong passwords and keep them safe.

Around 70% internet users know such things, but: What anti-virus to use? What precautions to take when virus is found?

Such things only half of this percentage know this. You should know every aspect; at least you should know what to do when virus is found, because this is all it takes. Secondly, if you do not use a proper anti-virus program, you will not be kept safe.

  1. Get original windows operating system; you can purchase it from microsoft.com, amazon.com, ebay.com or computer stores near you. Windows 7 is the latest Operating System from Microsoft. Do keep your windows updated.

  2. You should use a proper anti-virus program, an anti-virus program that is strong enough to defend from any type of attacks. These days there are different kind of attacks, like Trojans, malware attacks. And two anti-virus programs that are really powerful to fight these types of attacks are Kaspersky and nod32.. It’s true that Kaspersky can make your system run a bit slower, but it only happens if your system is not upgraded with the latest hardware, such as good ram and a new processor. Such programs are designed to run well on today’s fast computers. And the good thing about Kaspersky security is that, for Trojan, and malware, unlike other software where you have to use other separate software, it also includes these. It keeps scanning the incoming files from the internet, keeping your PC security at its best. And there are many new features in Kaspersky Internet security 2011.

Second is nod32; a good anti-virus program, which takes less memory on your PC. But Kaspersky is recommended. To increase your system security you can install Malwarebytes’ Anti-Malware too, which is the best tool to remove the malwares.

  1. Update your anti-virus software regularly. New viruses, worms, and Trojan horses are born daily, and variations of them can slip by software that is not current. Once you’ve installed the antivirus software, you will need to obtain regular updates from the manufacture that tells the antivirus software about new viruses and how to detect them. Most antivirus programs come with a year’s worth of updates, and you can usually set the software to either automatically download the updates, or display a reminder for you.

  2. Using a firewall, can keep you safe from hackers. Many personal computer operating systems include software-based firewalls to protect against threats from the public Internet. It is to prevent unauthorized access to or from a private network. Comodo firewall is very good, if you just want use firewall, but kaspersky already have a strong firewall.

  3. Scan your computer regularly. Allot time to scan your whole computer on scheduled days. This will allow you to easily detect a virus infection as soon as possible. You can easily delete the virus if it has not yet created any significant damage. Kaspersky provides different types of scans, one of them is “Critical area scan”, you can use “critical area scan” once a week, it take little time.

The Critical Areas Scan component consists of scanning the objects which are loaded at startup of the operating system, scanning the system memory, boot sectors of the disk drive, and the objects that have been added by the user.
