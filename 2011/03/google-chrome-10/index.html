Google Chrome 10 Is Here! More Faster, More Secure and More Easier

Mar 9th, 2011 | Comments

Google have just released the next stable version of their famous Chrome browser. Google Chrome 10 is coming with a lot of security fixes, new features to make things easier and safer and also some great improvements for the browser’s speed!

Actually, this is like the browser season… Both Internet Explorer 9 and Firefox 4 are about to be released in just few days… But Google was ahead of them. Actually, it’s not a miracle because Google have a very fast development cycle and they release new versions very fast!

Normally, Chrome is very popular for their fast java script handling. This have helped Google to make their browser one of the fastest in the world. The latest version of the browser is going to improve their java script performance by 66%! Their improved V8 JavaScript Engine is the key player of this performance boost.

Chrome 10 will also come with 25 security fixes where 15 are tagged as very critical! Theres a nother thing which improves Chrome’s security. They will now run the integrated flash player on a sandbox! Which means your browser will add a sandbox around your flash player preventing any malicious codes from infecting your machine though the player. There’s another big security risk, out dated plugins! Chrome have also decided to solve this issue by automatically disabling any outdated plugins.

Theres another big feature which you are going to love. Earlier, Chrome offered its options in a different window. But now, you can access your options through a simple browser tab just like visiting another website! Click on this link to go to the options : chrome://settings/browser (You need to use the latest version of Chrome for this link to work!). This new options tab is very simple and easy to use. You can also use the search box at the top to locate a specific setting without going through the whole page!

Actually, these are the major changes you are going to see on Chtome 10. To learn more, the best way is to use it! Normally, Chrome will get automatically updated on your system. But if you can’t wait, go to “Customize and control Google Chrome” (the wrench icon located at the top right corner of the address bar) and go to “About Google Chrome”. On Linux, it will get updated through normal system updates.
