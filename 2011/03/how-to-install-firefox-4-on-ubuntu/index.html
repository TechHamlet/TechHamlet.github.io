How to Install Firefox 4 on Ubuntu

Mar 24th, 2011 | Comments

This week, the Mozilla team released the final version of their latest Firefox browser. This latest browser comes with a complete re-design, and with support to the latest web technologies such as HTML 5 and CSS 3. For many days, we waited to get our hands on it. After releasing many beta versions, they have decided to release the final stable version on March 23. For Windows users, you can directly download the exe file and install Firefox 4. But, what about Ubuntu users?

As usual, you won’t get an easy update to Firefox 4 if you’re using Ubuntu. But, for Linux, there’s always another way! For all the past major releases, we used ubuntuzilla to install Firefox. But, now we have PPAs!
