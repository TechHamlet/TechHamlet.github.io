Firefox 4 Comes Into Ubuntu 11.04

Mar 28th, 2011 | Comments

Mozilla have recently released the latest version of their famous web browser and the Ubuntu team is getting ready to release the latest version of Ubuntu next month. For Ubuntu and most of the other Linux distributions, the default browser is Firefox. But the major drawback in this relationship is that Ubuntu will support only one major Firefox version until a newer version is released. So, for Ubuntu 10.10, we had to use a different repository for Firefox 4. But as usual, you don’t have to worry about installing it on Ubuntu 11.04.

The Ubuntu team have managed to put Firefox 4 into the latest nightly build of Ubuntu 11.04. As usual Ubuntu will keep this version of Firefox 4 updated throughout its life cycle..

But, now things are going to get changed. Mozilla is now planning to speed-up their development cycle and release Firefox 5, 6, and even 7 in this year. So, at least Firefox 5 and 6 will arrive before Ubuntu goes for the next version 11.10. If they stick to their old traditions and keep supporting new major versions of Firefox only after Ubuntu’s major releases, we won’t be able to see at least Firefox 5 on Ubuntu! But of course, we always have other ways of doing this, but it would be better if Ubuntu can support it by default! What do you think? Will they change their strategy?
