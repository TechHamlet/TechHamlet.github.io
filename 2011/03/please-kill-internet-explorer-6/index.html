From Microsoft : Please Kill Internet Explorer 6!

Mar 5th, 2011 | Comments

10 Years ago, Microsoft created a great web browser called Internet Explorer 6 which became very popular around the world! One of the main reasons why it became so popular is the fact that this browser was delivered free with Windows. Originally, Windows XP came with IE 6… Windows XP became a real hit… Still lots of people use this OS to power their machines. Windows XP still have some strength to power our machines, but not IE 6! Currently, IE 6 is named as a very old, out dated, and highly vulnerable web browser!

Now, the days of the Internet Explorer 6 is over. Microsoft have already released another two major versions of Internet Explorer and now getting ready for the third one. But still 12% of the world uses IE 6! When it comes to computers, we always advice people to upgrade to the latest version! When we release a new software, we always try to make it invulnerable to the existing threats. We also try to update it to face upcoming threats for some time. But, putting so many patches can also bring down the stability of a program. That’s why we go for new versions. Actually, IE 6 is 10 years old and it cannot face most of the existing threats!

When time pass by, everything change! We get new technologies to power our lives. The web have also received a lot of new updates in the past but IE 6 can’t support most of them. For example, Internet Explorer 6 won’t support any of the CSS 3 or HTML 5 technologies which have managed to change the way we use the web! So, if you visit to a site made from CSS 3 or HTML 5, it won’t work as expected on the old browser. So, the web developers have to write separate coding to overcome these problems. But now, even web developers have started to ignore IE 6!

Recently, Microsoft have launched a special website to ask users to stop using Internet Explorer 6. This site will show you how many people use this old browser. According to them, 12% of the world is using IE 6. The biggest portion comes from China which is about 34.5% while countries like Finland have only about 0.7%.

Now Microsoft needs you to stop using IE 6. They have also asked webmasters to add warnings on our sites to educate our visitors and let them upgrade to a new browser. Now Tech Hamlet have also added that warning and joined this cause! If you also have a web site, you can also join them.

Not only that, if you’re a normal user, you can still pass the message through places like Twitter and Facebook. What they need is to make sure that no one else will get hurt by using their own out dated software. Actually, we should appreciate Microsoft’s work and help them to pass the message around the world! Lets see how effective this new campaign will be! Lets see how many people will go for newer browsers!
