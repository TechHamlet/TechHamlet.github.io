Are You Happy or Sad About Me? Latest Firefox 4 Beta Wants to Know

Mar 9th, 2011 | Comments

Promising a faster, sleekier, and easier-to-use browser interface, the folks over at Mozilla Foundation have released the latest beta version of Firefox 4 just recently. It is available for downloading and testing on Linux-, Mac-, and Windows-based computers. The beta for the mobile version is available for Android and Maemo, too. The newest beta release includes fixes for more than 7,000 bugs discovered in prior beta releases.

Millions of developers, testers, and users around the world help in improving each and every version of Firefox. This latest beta version comes with the Feedback Add-on as an easy way for you to tell the developers why Firefox 4 makes you happy or sad. This is also the quickest way for you to send feedback about your experience with the beta version. You can opt in to—and easily opt out of—user studies conducted by Mozilla Test Pilot to gather data about how users worldwide use Firefox so that developers can use the data to fine-tune or further improve future releases of Firefox. Such user studies never collect your information without your permission.

Whether you want to help in testing this latest version, or you just want to check the upcoming new changes in the stable release, go ahead and download the beta version from the Mozilla website.

New Exciting Features

Here’s a quick rundown of the exciting new things to expect in the upcoming Firefox 4:

Tabs bumped upwards

Firefox has finally caught up with both Google Chrome and Opera. Both of the latter have had browser tabs at the top, much to the delight of browser users who need a lot of browser estate. In previous Firefox versions, you can achieve a Chrome-like appearance by using combinations of themes and extensions. In Firefox 4, the tab-at-the-top feature is built in.

Menu bar reduced to one button (Windows only)

Opera has had this feature for some time already. Firefox implements this in version 4. All menu items are now accessible via the Firefox button. This reduction minimizes visual distraction so that you can focus more on the content you are viewing.

Open-tab switching made easier

This is another speed improvement. Imagine having a lot of open tabs—say, 20 or 30 tabs (yeah, busy people are fond of keeping that many tabs open)—in your browser. Cycling through all of them with CTRL+Tab will still work. Firefox 4’s Switch-to-Tab feature makes finding that specific open tab easier and bring it to the fore. All you need to do is type that tab’s URL, or a part of the URL, and click on “Switch to tab” from the drop-down list.

Organizing open tabs with Panorama

Panorama is a new feature in Firefox 4. It allows you to create tab groupings in a way that is convenient to you. So, for instance, you can group tabs related to work, games, social networking, and so on. Another good thing about Panorama is that when you bring a tab group to the foreground, you will only see the tabs that belong to that group. This means less clutter, less distraction, and more focus on the task at hand.

Easier synching across many devices

Access your browser settings, passwords, visited pages, bookmarks, and open tabs from anywhere and using a variety of devices (e.g., another computer, an iPhone, etc.). Firefox Sync, formerly just an add-on, is now natively built into Firefox 4. Although your information is stored on a remote server, it is protected by end-to-end encryption, so your data remain safe, secure, and private.

Speed enhancements

Browser startup, rendering of graphics, and page loading times have decreased significantly. This means Firefox 4 has gone faster than its earlier versions. It also uses a brand new JavaScript engine called JägerMonkey.

More Improvements “Under the Hood”

There are more wonderful new things in Firefox 4, although you don’t get to see them because they work under the hood. Here are some of them:

  • Support for WebM and HD Video

  • More improved privacy

  • New JavaScript values (which means more efficient code execution and cleaner graphics)

  • Multi-touch Support

  • 3D on the Web through WebGL

  • Audio API

  • Retained layers (for faster scrolling of complex pages)

  • XPCOM module enhancements

  • Support for CSS3 features such as transformations and transitions

  • More flexible font features with OpenType

  • Integrated Form features and tools for easier form development

  • Support for HTML5

  • Web Console (experimental tool for analyzing dynamic web pages)

  • Easier development of add-ons via JetPack SDK and js-ctypes

  • Full hardware acceleration

  • Displaying SVG as images

  • Improved cache usage, security, management, and performance through Compartments for JavaScript

  • Better protection from plugin crashes or freezes

  • Automatic HTTP Strict Transport Security (HSTS) for better security and privacy

Firefox 4 is scheduled to leave the beta stage today, March 9, by which time Mozilla will make it available as a Release Candidate. The upcoming release of Ubuntu 11.04 (Natty Narwhal) set to be released on April 28 will ship with Firefox 4 as the default web browser, too. So, stay tuned and updated.
