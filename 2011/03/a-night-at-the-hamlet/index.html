Ubuntu Customization Guide : A Night at the Hamlet

Mar 14th, 2011 | Comments

Many people think that Windows have a better looking desktop than Linux. But, most of the time, this theory is wrong! Some Linux desktops might loo ugly at first, but that doesn’t mean you can’t r-design it! Well, in Windows, its hard to change the look of your desktop. Officially, they only support adding wallpapers and changing the color of the default theme. But, in Linux, you can re design the whole desktop, top to bottom!

Earlier also we wrote a small guide on how to change your Ubuntu desktop. That guide was named as “Purple Boy”. It installed a regular gnome theme, added some wallpapers, tweaked the panels, installed avant and also added some ring screenlets. Today, we are going to install a much more advanced gnome theme called Elegant Gnome, install wallpaper clocks and finally install the Guake terminal. I didn’t used conky on this guide because I had some problems installing it on my system at the moment. (It looks like many people have that problem, hope it will be solved soon). So, shall we start then?

First of all, lets install the Elegant Gnome Theme

  • Open the Terminal (Applications > Accessories > Terminal) and enter the following commands to install elegant gnome and nautilus elementry (required by elegant gnome) :

  • Now, to launch Elegant Gnome, go to the main menu, Applications > Accessories > Elegant Gnome

  • Select “install the pack” and click “OK”

  • Now, feel free to answer the questions they ask in the way you want

  • After the installation is complete, you need to logout from the system and login to restart the X server (Or just reboot your machine)

  • Now your desktop should look like this :

Now, to install the wallpaper clocks :

  • First, open the terminal and enter the following command to install screenlets :

  • Next, download the Wallpaper Clocks screenlet from here

  • Now, go to the main menu, Applications > Accessories > Screenlets. In the Screenlets Manager, select “Install”. Next make sure “Install Screenlet” is selected from the dropdown menu and hit “OK”

  • Now, locate the downloaded “Wallpaper Clock Screenlet” and click “Open”

  • After you’re done, select Wallaper Clock from the screenlet manager and tick “Start/Stop” and “Auto Start on Login” as shown in the image below :

  • Now close the Screenlet Manager. You’ll see a small clock on the desktop as shown below :

  • Drag it and place it wherever you want. Next, right click on it and go to properties. Next, select options. Untick “Keep above” and tick “Keep below” after that, close the window

  • Now you need to download a suitable wallpaper clock from here : http://www.vladstudio.com/wallpaperclock/browse.php?start=0&show=24&order=wcID&orderdir=desc. In here, I have used The Village Of Musicians. But you are free to choose any wallpaper you want.

  • Now again right click on the clock and go to “Install wallpaper clock”. Select the wallpaper clock (the .wcz file) which you have downloaded and click open

  • Again right click on the clock and go to Change Wallpaper Clock > My Wallpapers > and Select the wallpaper clock which you have just installed
