Google Chrome OS Coming Soon? Within a Week?

Nov 14th, 2009 | Comments

Today I saw many headlines saying that Google is about to release their Google Chrome OS within a week! This seems to be a leaked news, because when I go to the Google sites…. they were silent! No info about the Chrome OS…. this is going to be a Surprised Attack!

“The operating systems that browsers run on were designed in an era where there was no web. So today, we’re announcing a new project that’s a natural extension of Google Chrome — the Google Chrome Operating System. It’s our attempt to re-think what operating systems should be.” (Google Official Blog)

On July 7 The Google Official blog announced their new project Google Chrome OS. According to Google the new OS is going to change the history of operating systems… because the Chrome os is going to be the first OS which is going to be built targeting the web. The main target of Chrome OS is to get you to the web as soon as the system boots up.

“Google’s Chrome OS project, first announced in July, will become available for download within a week, we’ve heard from a reliable source. Google previously said to expect an early version of the OS in the fall.” (TechCrunch)

According to the TechCrunch the most awaited release will be happening within a week! But according to what i understood… this is going to be a beta release, which will be for testing purposes!

“Driver support will likely be a weak point. We’ve heard at various times that Google has a legion of engineers working on the not so glamorous task of building hardware drivers.” (TechCrunch)

They say that Google is still working on the driver support of the OS, which means… it might not work on some systems due to driver incompatibility! So, even if it gets released within a week, don’t try to install it by removing what you already have! If your no expert, forget about it….. If you a Pro and have some extra-time, install it parallel to your existing OS (We will tell you how to do this when it gets released! so stay tuned) and test it! Or just have fun!
