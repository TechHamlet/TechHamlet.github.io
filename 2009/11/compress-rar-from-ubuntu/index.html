How to Compress Into RAR Files From Ubuntu

Nov 16th, 2009 | Comments

RAR is one of the most widely used compression methods in the tech world. For most of the people who use windows, when we say compress… its to compress with RAR. Unlike other compression methods, RAR provides a good set of features like SFX archives, spliting the compression into several parts, good encryption methods, tight compression, etc.

By default Ubuntu doesn’t allow us to create RAR archives. But, does that mean its impossible? Not at all…. its very easy… lets see how to make RAR archives in Ubuntu :

  • First open the terminal and type sudo apt-get install rar

  • Next right-click on what ever you want to compress and click on “Compress”

  • Finally select RAR as the file extension (as shown in the image below) and click on create. (if you want split the archive into multiple volumes or if you want to use encryptions click on “Other Options”)
