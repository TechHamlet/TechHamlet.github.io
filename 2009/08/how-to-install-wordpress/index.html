How to Install Wordpress

Aug 12th, 2009 | Comments

These days blogging has become a very common thing, many people have their own blog running. People use many software to make these blogs, you don’t have to know HTML or PHP to run a blog, these software makes our lives easier!!! Among these Wordpress is very popular, because its simple,Â  easy to use and easy to learn. So we thought of teaching you how to install wordpress on your server.

(Version we used 2.8.4 (all subversion under 2.8 must work))

Things you need :–

  • A server (your own server or an account in a public web host like)

  • A database (can make one from the c-panel)

  • Wordpress files (Download)

1) Extract the Wordpress files you downloaded

2) FTP into your server and upload those file to the server (upload to the root folder or if you want your blog in a different folder make the folder and upload the files into it)

3) Direct your browser to your domain (www.your-domain.com) or if you installed in a sub folder direct to that folder (www.your-domain.com/folder). Then wordpress will tell you to create a configuration file as in the screen shot :–

4) Click on Create Configuration File. Then you’ll get a screen as follows :–

5) Click on Lets Go. Then in the nest screen you have to fill in the details of the database you have created. (Note : Change the table prefix for better security, use anything you like (abc , you))

6) After you fill in double check whether the data is correct, the click on submit. Now that you have given the data, you should allow wordpress to run their installer. In the next screen click on Run the install.

7) In the next screen enter the name of your blog (Blog Title) and the e-mail your willing to use for the blog. Then if you want others to know about your site, if you want other websites to link to your site check Allow my blog to appear in search engines like Google and Technorati. (don’t do this if you want to keep your blog private, only you and your friends can see it). After filling click on Install WordPress.

8) Now wordpress will finalize everything and in the nest screen you will be given a user name (default : admin) and a randomly generated password (Note : Copy this password and save it, you need it to login to wordpress). After you copy the details click on Log In.

9) Now in the next screen give the user name and the password you got from the previous screen and Log In!

10) If everything went fine then you should be able to login to your wordpress dash-board (Note : Remember to change the password as soon as you log in)

Ok folks thats it!!! Hope you got it without messing up…i’ll soon write some more about how to work with wordpress and about good plugins…so until then happy blogging!!!
