Install Another OS Inside Windows and Use It Like a Software

Mar 21st, 2010 | Comments

Do you know that you can install any number of operating systems inside your own desktop and you can minimize and keep them just as normal programs? If not… its time to learn! Today, we’ll see step by step, how to install and run another operating system right inside your windows desktop!

Virtual means not real! I hope all of you know that! Today, what we are going to do is, install a virtual OS inside windows! To do this, we are going to use a software called VirtualBox, which will help us to make a virtual environment to install our guest OS. In this environment, we can install Linux, Windows, BSD or Sun Solaris!

Why Do We Use this?

  • To test an OS before actually installing it.

  • For learning purposes. If you want to learn how to configure a web server, without buying a separate machine for that, we can use a virtual OS. You can also create a virtual network of computers.

  • To test our software. (For big software, better not to use this)

  • Finally, to show your friends that your a geek!

Enough talking… Lets see how to do this :

  • First, Download and Install Virtual Box (Download link : http://www.virtualbox.org/wiki/Downloads)

  • Next, open VirtualBox

  • Now, to start creating a virtual machine, click on “New” as shown in the image :

  • First, you’ll see a welcome message (which most of you will ignore). Click on “Next”

  • Now, you’ll see a screen as below :

  • In here, first give any name to your Virtual Machine.

  • Next, select what your going to install. For example, in the above screen-shot, we have selected, “Microsoft Windows” as the operating system and “Windows XP” as the version.

  • After your done, click “Next”

  • Next, you need to give the maximum amount of Memory that your going to allocate for your Virtual Machine. (Do not use too-much, try to use a middle amount). After your done, select, “Next”

  • Now, we need to create a new “Virtual” hard disk. In here, make sure “Boot Hard Disk (Primary Master)” is ticked. Then select “Create new hard disk” and click on “Next”

  • After that, you’ll be greeted by a welcome message. Click on “Next”

  • Select, “Fixed-size storage” and click “Next”

  • Now, you’ll see a screen as below :

  • Give the location which you need to place your Virtual Disk (need to have a lot of space)

  • Next, give the amount of hard disk space which your going to allocate for this disk. (10 GB is good for any OS, But for Windows 7, you’ll need about 20 or 25)

  • After your done, click “Next”. Now, click on “Finish”.

  • Now, it will create your Virtual Disks (which will take some time) and give you a final summary. In here, click on “Finish”

  • First, you need to install the guest OS. To do this, you need to have an installation CD or an ISO image of that OS.

  • To add the installation CD, In Virtual Box, select the created Virtual Machine and click on “Settings”.

  • In the Settings window :

1) Go to “Storage”

2) Select “Empty” under “IDE Controller”.

3.1) Now, if you want to use a CD in your CD drive, select “Host Drive” under “CD/DVD Device”

3.2) If you want to add an ISO image, click on the icon shown as “3.2”Â  in the image below. After that, click on “Add” and give the path to your ISO image, finally click on “select”

  • After your done, click “OK”

  • Now your ready to go… Select your virtual OS and select “Start”. Now you can install and use your Virtual OS!
