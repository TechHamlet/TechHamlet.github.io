Have You Seen the New Google Image Search?

Jul 31st, 2010 | Comments

The Technology is some thing that evolves very fast! Something new comes every day! If there’s something old.. it won’t exist for long. I think you have seen the latest redesign of Google Search and might have even read about it from us (Have You Seen the New Google Interface?). That day they have made some big changes to their famous search engine by adding more new features and a cool new look. Today they have given a brand new look for their image search!

I think this is one of the biggest changes that Google have made to its image search after a very long time. On the July 20th, they announced about this great update through their official blog. But yesterday only we got the chance to experience these changes. This new interface have a lot of new changes which will be able to give you more relevant results more easily. Some of these features are :

  • Packing all the resulting images tightly. Earlier when we search through the image search, they gave us the results with the details at the bottom so at least 40% of the page is taken by the image details. This new interface will display only the images nicely packed all over the results page making it more easier to find the relevant image.

  • Earlier, the image search results were divided into several pages so that we have to navigate to the next page if we didn’t find the result in the current page. The new interface will load all the images in one page! According to Google, now one page sows upto 1,000 images and if you want to view more, you can select “Show More Results” button at the end of the page, which will load more images in the same page.

  • Now where did they took the image details? Just hover your cursor over an image and you’ll get the answer. When you hover over an image in the results page, the image will get zoomed out and will also show the details of the image.

  • When you click on an image, earlier it will take you to a page where you’ll get a toolbar by Google showing the image and the details at the top and the web page where you have the source web page at the bottom. But now.. it will take you to a page where you’ll get the Google toolbar with the image details on your right and the web page on the left. You’ll get the image zoomed in the middle of the web page. The page will be darkened to highlight the image.

I think these new changes that they have made after many years is something that should be available for a better image search. I also have faced a lot of trouble finding good images for my blog posts and also for my designs. But now I think things will be much more easier and fun getting the right image for your job! What do you think?
