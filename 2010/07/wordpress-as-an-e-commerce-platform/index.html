The Why and How of Using Wordpress as an e-Commerce Platform

Jul 28th, 2010 | Comments

[ad#Roko Nastic - Lead-in AdSense Text]

Wordpress started in 2003, and has quickly become one of the most popular self-hosted blogging and CMS platforms. Traditionally, e-Commerce websites required a large amount of time and technical knowledge to create, but Wordpress has made creating an e-Commerce platform easy. Let’s take a look at why Wordpress is a great e-Commerce platform and how it is done.

Why Use Wordpress as an e-Commerce Platform

1. It’s Free

While there are some commercial options available, Wordpress is a free platform that has proven to be extremely customizable and easy to use. It is now being used by everyone from first time bloggers to Fortune 500 companies. Since it is free, you can avoid licensing fees, upgrade fees, support fees, and even installation fees.

2. Flexible and Customizable

One of the reasons that Wordpress continues to gain popularity points is because it is easy to customize and flexible enough to meet everyoneâ€™s needs. In fact, you can change the entire layout by simply switching themes. Additionally, you have the power to modify every theme to make it look exactly like you want it to. There are also a number of excellent free and commercial plugins that allow you to add a variety of features with a click of a button. With so many ways to extend the functionality of a Wordpress website, your options seem endless.

3. Search Engine Friendly

Wordpress has a number of excellent search engine friendly e-Commerce plugins. This will ensure that all of your products get indexed and show up in the major search engines. Wordpress also makes it easy to have keyword friendly URL’s which can boost your SERP’s.

[ad#Roko Nastic - MidPostAd]

How to Use Wordpress as an e-Commerce Platform

Wordpress does not come out of the package with e-Commerce abilities, however by simply adding one or two plugins you will be able to create an online store and process payments from any gateway. It is important to note that not all e-Commerce plugins include a payment processor, which means you may require two plugins instead of one. Additionally, it is important to make sure that you choose a e-Commerce theme. This will ensure that the plugins display everything correctly, as well as give your website the look and feel of an e-Commerce website.

1. Best e-Commerce Themes

There are a number of great e-Commerce Wordpress themes, however some that have been more popular and easier to customize than others. . is a very popular theme because of its built-in functionality. Not only does it integrate with more than 20 payment gateways, but it is completely browser based so that it will work from everywhere. It features 20 different shopping cart designs and Amazon, eBay, CSV, Datafeeder import abilities, along with simple Google Analytics and Adsense integration.

Vanilla Cart is another popular theme because it is designed minimalistically so that you can easily customize it. It was designed specifically to be integrated with the WP e-Commerce plugin. Another popular theme that was designed to be integrated with the WP e-Commerce plugin is CraftyCart. Another great WP e-Commerce compatible theme is Storefront. Storefront was designed with existing WP e-Commerce code already integrated with some custom coding to give you the best results possible.

2. e-Commerce Plugins

Most recognized plugin is definitely WP e-Commerce. This plugin is a fully featured shopping cart application that allows you to sell not only products, but services as well. It was designed as a Web 2.0 application to make it not only usable, but also looking great. This plugin is often referred to as the most complete and powerful Wordpress plugin available.

Yak Shopping Cart is another solid Wordpress e-Commerce plugin. It is a very simple shopping cart plugin that allows you to associate products with specific posts. Because of its simple design, it is very easy to modify and customize to fit your needs.

Quick Shop is another unique Wordpress plugin. It adds a sidebar widget that will show the user what they currently have in the cart. This plugin also has a number of payment gateway options as well as TinyMCE button to make adding and deleting products from the shopping cart simple.

It is important to note that many, though not all, shopping cart plugins have certain payment gateways specifically integrated with it. This is important to keep in mind because if you use Google Checkout and the shopping cart plugin that you want to use isn’t designed to be compatible with it, then making the necessary modifications may be difficult. Additionally, some e-Commerce plugins have no payment gateways pre-configured, which means that you will need to do this on your own.

Wordpress has proven to be an excellent e-Commerce platform for online stores of all sizes. However, it is important to choose a theme that is compatible with an e-Commerce platform. Some themes are fully functional Wordpress store, whereas others are meant to compliment specific Wordpress plugins. It doesn’t matter which option you choose, they can all create a great looking, functional online store.
