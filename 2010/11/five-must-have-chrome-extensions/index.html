Five Must Have Chrome Extensions

Nov 20th, 2010 | Comments

It’s been a while that chrome started getting more and more productive extensions from various developers. Most of the chrome users tend to use it without having installed any extensions not knowing what chrome is really capable of. Not only extensions boost your productivity but also it will enhance your browsing experience and let you havc some fun too.

Here are five best chrome extensions that every chrome user must use. The best thing about the chrome extensions is you don’t have to restart you browser to make them function!

  1. Chromed Bird

It’s really a pain to fire up the twitter homepage or to run a desktop client, just to share what you mind holds at the moment. This extension allows you to tweet,shorten URLs and many things with just one single click. An icon of a bird will be showed up when it’s installed and it’ll turn into red when it gets a new update. The interface is clean and easy to use, giving access to your twitter stream, direct messages, favourites and so forth.

Chromed Bird

  1. Xmarks

Syncing bookmarks has never been this easier without Xmarks. If you are frequent flyer or someone who’s always on-the-go then it would be quite handy to sync your bookmarks without having to recall them. Xmarks uses cloud based servers to sync your bookmarks. To get back your bookmarks on a new machine, you simply have to install the plugin again and just click on “Synchronise Now” It’s that easy!

Xmarks

  1. Apture Highlights

Apture Highlights is a free tool, built from the ground up to let you take the power of Google search, and the richness YouTube, Flickr, Twitter, and Wikipedia with you to any site. Just highlight a phrase on any site to reveal the web’s best content without ever leaving the page. Fast, powerful, and fun.

Apture Highlights

  1. Facebook for Chrome

Eventhough this is an unofficial extension, it allows you to check your inbox, wall, notifications in one place. It will automatically notify you when you have new notifications and unread messages. The latest version sports cool features like remembering what tab you had open last, auto-expanding place to write status updates and so on.

Facebook for Chrome

Source Code

  1. Stylebot

This is very useful for web developers, but even if you don’t know web developing this for you too. Basically what it does is, it allows you to change the appearance of any website by altering the CSS. The real deal is that you don’t have to know CSS at all! It provides you a nice interface to tweak all the appearance settings in one place.
