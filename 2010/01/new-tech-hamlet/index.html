Tech Hamlet : A New Name, a New Address and a New Look

Jan 8th, 2010 | Comments

What happened to Earth Hacks? What happened to earth-org.com?… I know, these days you might be trying hard to find answers for these question! In simple terms, we just renamed our old site, and gave it a new address! Lets see how and why we did it…

Why did we change our site?

  • Bad Domain name :

Our old domain name “earth-org.com” is not a good domain name. First reason is the dash “–” . People normally don’t use dashes and stuff for domain names, because a domain name should be very very simple! The second reason is the name “earth”…. Actually, when we say “earth-org.com”, people think that this is a site which tries to help the environment or some thing like that!

  • Bad Name :

“Earth Hacks” is not that bad… but when it comes to search engines, things are different! When we search for “Earth Hacks” in Google, our site comes to the 5th place! For about 3 months, the story is the same.

  • Sub-domain :

Earth Org is supposed to do many projects related to information technology. So at first we thought of putting all our projects under the main domain name. But now we understand that if we give a separate domain name for each project, we can get more better results!

How did we change?

  • Spent some time to get a good name :

These days finding a good domain name is not an easy task, most of the domain names are taken by cybersquatters. So we spent some extra time for that, and came up with techhamlet.com. Its not only a good domain name, but it also have a good meaning :

The concept Global Village brings everyone in the world closer, so that we can communicate with each other like we were in one village! ButÂ  the concept Tech Hamlet will bring everyone much more closer… now its not a village, its a hamlet (a hamlet is a small village)!

  • Next we transferred our old site to the new one with new features :

This phase took some time because we had to face lots of problems while dong this. First we had to decide which method to use for the transformation. First we tried to do it from Export and Import functions in Wordpress, but this messed up the whole site! So we have to do it from the databases.

  • Finally, had to update the links and polish things up :

Next we updated the links in search engines, and some other sites to point to the new site! And also added a permanant redirect from our old site to the new one, so that any one visitng the old address will be automatically redirected to the new one!

What did we gain from the change?

Actually we gained a lot from this change. First of all, we got a new and a meaningful name : Tech Hamlet! Next we got an indentation domain name (we means the blog, because earth-org.com also belongs to us).

We had some small problems with our old theme, so we decided to change it while transferring to the new site. So we got a new theme with more great features!

The most important thing is that, unlike our old site, we have already hit the first place in Google search! The old blog tried to get this place for months, but the new one, did it in several days!

Now, as our loyal readers, what do you think about our Big Change?
