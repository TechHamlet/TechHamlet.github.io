How to Turn Off Annoying Warning Messages on Windows 7?

Jan 19th, 2010 | Comments

The User Account Control Settings of Windows might become a bit annoying. When ever we try to install any program or do an update to the system, UAC warning message pops out of nowhere! So, do you know how to get rid of this? If not, lets see how…

Why do we want to turn UAC off?

  1. As I said earlier, it might be irritating when it pops out every time.

  2. Next, it might prevent us from changing some settings like enabling write caching.

Why do we want to keep UAC on?

Do you know why we have User Account Control in windows? Its there to prevent malware from being automatically installed on our system. It works like this… Actually most of the malware in the world were installed without the consent of the user. But, if you have UAC turned on, you can reduce this problem, because it asks you when ever something tries to get installed on your system. Of course it won’t stop all the malware attacks!

Now, if your ready to take the risk, lets see how to turn it off :

  • First, go to Start > Control Panel

  • Then click on “User Accounts and Family Safety”

User Accounts and Family Safety

  • In the next screen, select “User Accounts”, after that click on “Change User Account Control Settings”

  • Now this is the place where you can turn UAC off :

User Account Control Settings
  • There are 4 options :

1) Always notify me

2) Default - Notify me only when programs try to make changes to my computer

3) Notify me only when programs try to make changes to my computer (do not dim my desktop)

4) Never notify me

  • To switch UAC off you need to select the last option : Never notify me

After reading the post, tell us what you did… Did you turn UAC off or did you turn it on after identifying the risks?
