Update Your Browsers : Mozilla Releases Firefox 3.6.9

Sep 8th, 2010 | Comments

Firefox 3.6 is the latest stable build of the famous Firefox Browser! Recently, the Mozilla foundation have started their work on the next stable build, Firefox 4.0.. Even when they are closing in to the release of Firefox 4.0, today they have released another update to Firefox 3.6!

The latest Firefox 3.6.9 is released mainly to add support to X-FRAME-OPTIONS which will allow the webmasters to specify whether the browser should render the page in or <iframe>.

Sites can use this to avoid clickjacking attacks, by ensuring that their content is not embedded into other sites.

Not only that, it will also fix about 14 other security issues and several stability issues which are meant to protect the normal users. So, your requested to update your browsers as soon as possible!

How to Update Firefox :

If you already have Firefox :

  • Most of the time, Firefox will automatically download the new update and notify you.

  • If not, you can do it manually by going to Help > Check for Updates in Firefox.

If you don’t have Firefox :

  • You can download the latest stable version of the browser from here : http://www.mozilla.com/en-US/firefox/firefox.html
