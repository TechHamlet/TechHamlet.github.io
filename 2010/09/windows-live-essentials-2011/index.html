Windows Live Essentials 2011 Is Here!

Sep 30th, 2010 | Comments

We were eagerly waiting for Windows Live Essentials 2011 to be released, test and review it for our dear readers as soon as they release it! It contains a bunch of programs like Windows Live Mail, Messenger, Photo Gallery, and the renamed Windows Live Mesh. As Microsoft promised they have done a lot of improvements to deliver a greater user experience than before.

Users have the option of installing the entire suite or only choose certain elements, thanks to the custom install option from the compact installer.

System Requirements

  • Operating system: 32- and 64-bit editions of Windows Vista Service Pack 2 with the Platform Update for Windows Vista, Windows 7, Windows Server 2008 with Service Pack 2 and the Platform Update for Windows Server 2008, Windows Server 2008 R2.

  • Processor: 1.6 GHz or higher

  • Memory: 1 GB of RAM or higher

  • Resolution: Minimum: 1024 × 576

  • Internet connection: Internet functionality requires dial-up or broadband Internet access (provided separately). Local or long-distance charges may apply.

  • Graphics or video card: Windows Live Movie Maker requires a video card that supports DirectX 9 or higher and Shader Model 2 or higher.

New Features and Improvements

  • Faster Messenger.

  • Bing Maps Integration.

  • Better integration with Office.

  • Messenger Integration with Facebook.

  • Rapid advances in the photo gallery.

  • Efficient Video Chatting.

…and many more!

So hurry up and enjoy Windows Live Essentials 2011 yourself! Click on the following link to grab the installer.
