Unleash Your Game Development Skills With Unity 3D

Sep 17th, 2010 | Comments

Having an integrated graphical environment is the primary method of game development. Unity is here to fulfill that requirement.

Unity is a multiplatform game development tool, designed from the start to ease creation. A fully integrated professional application, Unity just happens to contain the most powerful engine this side of a million dollars.

Major Features

•

Professional FPS controller ready to drop in (and tune)

•

Streamed loading for the Unity Web Player

•

Unity asset server / asset source code version control

•

Cross-platform web player content, the Unity Web Player is available for both Mac OS X and Windows users and works with all browsers

•

Standalone executables for both Mac OS X and WindowsObject-Oriented Design, Plug-in Architecture, Save/Load System

•

Mac OS X Dashboard Widgets

•

iPhone Publishing is available as add-on product

•

Streaming Asset Bundles: the ability to stream in any asset (terrain, mesh, etc) into the game.

•

Asset Processing Pipeline: in larger projects, customizing the art pipeline to avoid repetitive steps becomes crucial. You can now complement Unity’s famous asset importing pipeline with any additions you need to make it truly flow.

The graphics engine uses Direct3D, OpenGL, OpenGL ES (for iOS) and proprietary Nintendo Wii APIs.  Assets loaded into Unity and are automatically imported, and are re-imported if the asset is updated. Unity supports integration with3ds Max, Maya, Blender, Cinema 4D, and Cheetah3D.

Deployment

Unity supports a wide range of publishing platforms, all available at the push of a single button. You can publish standalone builds for Windows 2000/XP/Vista/7 , Mac OSX, Nintendo Wii, iPhone/iPad and for the Unity Web Player. Either way you’ll be able to dazzle your audience!

Shaders

Unity comes with 40 in-built shaders from the simple to the very advanced. If you aren’t satisfied with them, you can write your own shaders!

3D Audio and Video

Play back video on any surface in your game, or directly in full-screen. Sync or syncopate audio and video. Place positional audio-sources anywhere. Tune distance attenuation and volume parameters for the most immersive audio experience possible.

Advanced Physics

Unity contains the powerful NVIDIA PhysX next-gen Physics Engine. More over, unity supports full Rigidbody physics, hinges, springs, ball-sockets, character limbs, and the fully-customizable configurable Joints. So creating a blazing spots-car has never been easier.

So you might ask, how to get started? It’s very easy. Download Unity from the following link:

  • http://unity3d.com/unity/download/

And here are some Unity 3D tutorials :

  • http://unity3d.com/support/documentation/Manual/Tutorials.html

  • http://www.unity-tutorials.com/

  • http://www.design3.com/unity
