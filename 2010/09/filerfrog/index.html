Go Beyond the Windows Explorer With FilerFrog

Sep 24th, 2010 | Comments

Feeling bored with the default windows explorer? Then try FilerFrog and strengthen it up! FilerFrog works simply as a shell extension and provides you an array of  features like resizing images, customizing them with a logo, encrypting/decrypting files and many more!

Downloading FilerFrog

You can obtain a copy of the installer from here : FilerFrog

Installing and Using FilerFrog

Installing the program is as the same usual way of installing any ordinary windows application. So it’s a piece of cake! After the installation process it will be integrated with your windows explorer.

The specialty that lies beneath this nifty application is that the functions it provide are completely file dependent. That means if you click on an mp3 music file, all the image manipulation functions will be disabled.

The image manipulation functions are pretty handy (especially for me!) so that you do not need to run any other programs to manipulate them such as resizing.

Organizing Files

All what you have to do is just click on the image and right-click, select FilerFrog→Image Manipulation.

One interesting option in this menu which is designed for more experienced users is called Touch. Activating it has the effect of changing the creation/modification and last access time and date with the current one.

Security features as encrypting has already been added and it encrypts your data using RC4 stream cipher.

Encrypting Data

Decrypting your data is as simple as double-clicking the file you want and entering the password you put before.

One of the major drawbacks of FilerFrog is that it does not allow you to manage shortcuts and you’ll have to use the Favorite function instead.

Overall, FilerFrog is a handy and a neat shell extension which can save your time in day-to-day file handling tasks and image manipulation tasks.  To be honest, it’s quite good for a normal user, but you are a power user you won’t be much surprised about the features it injects into your windows explorer.
