Updated : What Shall I Do After Installing Windows 7?

Dec 22nd, 2010 | Comments

Ok, Now.. After installing Windows 7 on your system.. Do you think that its wise to start your work right away? If you do like that.. After some time, you’ll have to re-install Windows again due to viruses and bad system maintenance! So.. what should we do to keep a healthy system? What software do we have to use? What kind of tweaks do we have to use?

After installing Windows 7on your system, there are several things that we have to perform before starting our day to day work with it.

What to do and why :

  1. First we need to make sure the system have the latest updates so its more secure and stable. Windows release most of these updates to patch security holes and correct system errors that they discover after releasing their products. Think that there’s a big virus attack and that virus is using a small hole in your windows system to attack your PC. After that, to prevent this virus from entering your system, Microsoft will release a patch to cover that hole. All the systems that have that patch will be saved and the others…. You know what!

  2. After that, turn off autorun and autoplay. There are lots of viruses crawling out there willing to attack you through removable disks like USB drives and CDs. The key feature which allows these viruses to spread is the autorun feature. Autorun will automatically execute anything in the autorun.inf file located in the removable disks. So, virus developers have used this to spread their viruses even to machine which doesn’t have an internet connection!

  3. Next we need to make sure that our drivers are up-to-date. To get the maximum performance out of your hardware (specially graphics (VGA) cards) you need to install the latest drivers on your system.

  4. After installing the drivers, we need to tell your Windows 7 system that we have installed new drivers. Normally when you install Windows 7, it will adjust its settings according to your current hardware and drivers. If you don’t have drivers for example the VGA, the system will think that you have an old VGA and disable some features like windows Aero. To re enable these features, we need to refresh the “Windows Experience Index”.

  5. Most importantly, you need to build a good protection to your fortress which is in the middle of an enemy territory! Its right.. you can never say that your system is completely protected even if your not having an Internet connection! Other major methods you can get viruses are, USB Pen drives and CDs. So, its very important that you use a good firewall and an anti virus system. Note : Use of cracked (illegal, pirated) software is not recommended, because most of those cracked software contain hidden viruses.

  6. Add more features to your windows explorer. If you want to do more with your explorer, then you can add features like resizing images, customizing them with a logo, encrypting/decrypting files and many more using a small free software!

  7. Increase the speed of your copy pasting and make sure you copied everything without any errors. Sometimes when you try to copy files, you might get various errors. Sometimes, everything will go fine from the outside.. But when you try to access the files.. boom! Don’t worry, we have a cure for this disease too!

  8. Next we need to install some system maintenance software and give an initial cleanup. After using your system for some time, you might notice that its performance is gradually going down. This is because of junk files and disk fragmentation caused by other software. When these problems occur, we need to correct them out to make sure that your system runs smoothly!

  9. Finally, get some cool themes to decorate your desktop!

How to do :

1) After installing Windows 7, connect to the Internet (Some times you might need to install the network drivers)and go to Main menu > All Programs > Windows Update. (Note : If you have a network, make sure that all the other systems are removed form the network. Another major method which you can get malware is through a network.Malware like worms spread through a network and infect all the systems which are not protected. So, until we install the security software in your system, make sure that the system is alone in the network.)

On Windows Update, select “Check for Updates” as shown in the image and install all the updates when it finish checking. After installing the updates, it will ask to restart the machine. Go ahead and restart the machine to complete the updates.

2) Follow this tutorial to disable autorun and autoplay : How To Stay Away from Viruses Spreading Through AutoRun!

3) Now insert the driver CDs that came with your system (Specially the Mother board and the VGA) and install the drivers. Before that, if you can go to the manufacturers website of your hardware, go there and download the latest drivers for your system. Remember to install the drivers one at a time and restart after each install. If your using the CD, it will sometimes do this automatically (Intel does this).

  • First install the Chipset Drivers and restart

  • Next install the Network and Sound Drivers and restart (You don’t have to restart after installing the network drivers. So, install it first and then go for the sound drivers. Sometimes you might need to install the network drivers at first to connect to the Internet (for the updates). If you have already done that, skip it at this stage.)

  • Next install the VGA drivers and restart.

  • Finally install the other drivers like printers and cameras.

4) Next, to refresh the Windows Experience Index, go to your desktop and right click on “Computer” icon and select “Properties”. In the Properties window, select “Windows Experience Index” as shown in the image.

In the Windows Experience Index windows, select “Re-run the assessment” or “Refresh” as shown in the image below.

After Re-running or Refreshing the assessment, its better to restart your machine.

5) Now its time to build your defenses! Choose a good anti virus system to protect your PC. I’ll suggest Kaspersky Internet Security or BitDefender Internet Security. These guys comes with an almost complete protection with an Antivrus and a Firewall. If you don’t have money, my advice is, never go for pirated or illegal copies because its like installing a malware which will always say Hi to other malware! So, if you don’t have money, go for free products which also have a very great protection. Some are also able to beat the big guys. For example, Microsoft Security Essentials is said to have the best false positive detection rate. Some top free antivirus systems are :

  • Microsoft Security Essentials

  • Avast

  • Avira

  • AVG

Most of the free antivirus systems don’t come with a Firewall, so need to install a firewall separately.

  • Comodo Firewall

  • Online Armour

  • ZoneAlarm Free

After you install the security systems, don’t forget to update them. Always keep these software updated. Most of them will have automatic updates. If not, better to update them daily.

6) You can learn how to add more features to your explorer by following this tutorial : Go Beyond the Windows Explorer with FilerFrog

7) Learn how to increase the speed of your copy pasting and prevent any errors : TeraCopy : Copy Your Files Safer and Faster

8) Next is the time to install system maintenance software. For this, its almost useless to pay huge amounts because there are lots of free system maintenance tools which can do a great job on maintaining a smooth PC.
