A Great Compression Tool After All : PowerArchiver 2010 Free

Oct 28th, 2010 | Comments

﻿Power Archiver 2010 Free is a  great replacement for other archiving tools that are available for Windows. There are two major versions such as Personal(Free) and Commercial. I prefer choosing the personal edition as it fulfill all the file compression and related stuff of an average user. It offers Explorer integration, multiple encryption methods and advanced compression algorithms to name a few.

New ZIP engine fully supports ZIPX archives (LZMA, JPEG, PPMD, WavPacl, BZ2 format support) and makes PowerArchiver the only compression utility fully compatible with WinZip 12/13/14. Other ZIP/ZIPX related features include full Unicode support, AES support and unlimited file size/number support. Power Archiver supports the open source 7z format which is capable of 50% of compression than the zip format. The most unique feature that intrigued us was its optimized compression mode in which case it will automatically select the best format!

Key Features of Power Archiver 2010 Free

  • PowerArchiver supports following formats: ZIP, RAR, 7-ZIP, CAB, ACE, HA (LZH), TAR, GZIP, BZIP2, BH, XXE, UUE, yENC, and MIME (Base 64), ARJ, ARC, ACE, ZOO plus ISO, BIN, IMG and NRG disc image formats.

  • PowerArchiver has plugin system which can be used to add more formats support (msi, chm, etc).

  • Fully integrated shell extensions

  • Backup Wizard

  • Encrypt/Multi Extract/Batch Compress/Archive Converter/Repair ZIP/ tools

  • ZIP/CAB/7-ZIP SFX archive wizard

  • Added support for extracting LZMA2 archives

  • Updates of direct skin engine

Here are some screenshots that won’t posses further clarification.

As you might have already guessed its default interface is similar to that of WinZip. But you can get a fully-fledged UI with the power user mode!

Pro version of PowerArchiver adds even more features: Professional level backup utility that lets you backup your files to 3 destinations as well as ftp/email or burn to the disc, FTP client for easy transfer of files via FTP and integrated standalone Burner utility which lets you burn data CD/DVD/Blue-Ray discs, rip your discs to ISO images and even use Virtual DVD tool to mount ISO files. So if you are interested, you can go ahead an buy it!

Hope this will give a solution for your compression tasks in Windows! Fire up your comments if you have a better compression utility for Windows than PowerArchiver.
