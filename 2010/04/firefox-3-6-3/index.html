(Firefox 3.6.3) Mozilla Fixes Another Critical Security Issue in Firefox

Apr 2nd, 2010 | Comments

Yesterday, Mozilla have released another fix for their latest Firefox 3.6! Earlier they fixed some security and stability issues… This time, they have fixed a critical security issue which will allow remote code execution!

This is the second fix that Mozilla have released to patch Firefox 3.6. As I said earlier, this time they have fixed another critical security issue which will potentially allow remote code execution on your browser! This time, we were unable to give you more details because Mozilla won’t allow us to view the details of this bug (bug 555109)

Can you remember once we wrote about the Pwn2Own contest? (Pwn2Own : IE – Down, Safari – Down, Firefox – Down, Chrome – Not Tested) In that contest, Nils have brought down (hacked) Firefox on Windows 7 using one of its bugs! I think, in this patch, Firefox might be trying to fix that security issue!

Now, what you got to do is, update your browser as soon as possible! Always stay up-to-date!

How to update :

  • Open your Firefox browser. In the menu, go to “Help” > “Check for Updates” or “Apply Downloaded Updates now”

  • Or download it from here : http://www.mozilla.com/en-US/firefox/personal.html
