GIGA Tweaker : Tweak Your Windows Machine for Ultimate Performance

Apr 10th, 2010 | Comments

Using a high-tech OS like Windows 7 these days involve a lot of complications to an amateur user. To simplify the matters we have many tweaking tools that help us run our PC neat, clean and fast. The newest additions to this world of tweaking applications is “GIGATweaker”. An app that’s shaking up the web with many positive rave reviews. The perfect tool for tweaking windows 7 some tend to say. So Tech Hamlet decided to test this trendy tool and tell our loyal readers about the pluses and minuses of this newest addition to tweak world.

To begin with the application is comparatively smaller in size compared to most apps around. The setup files weigh nearly 2MB and when fully unpacked and installed, it comes toÂ  about 5MB. So light weight ensures smooth and uninterrupted running needless to be said.

After installation and creating a recovery point it should be said that this app impressed us. It seems to be a very effective control panel to make alterations and changes that matter most to the OS and ensuring smooth sailing without the ultimate “CRASH” hampering your life. Multiplicity of options and tweaks available might arouse confusion in an amateur but when gotten used to it will be easy as a piece of cake. Two panel interface is definitely a plus compared to some other tools in market that keeps opening up multiple windows for every tweaking task that you click on.

So, summing up the key tweaking features available :

  • Log on and automatic log on features plus display of messages before log on

  • A management feature for how the PC should act in case of a system failure and error management

  • Common security options in the likes of user account control, action center notification management and complete control over windows update.

  • Settings related to files. Designation of default software to open unknown files, auto play management and check boxes to enable and disable low disk space notifications and positioning of driver letters.

  • Disabling task manager

  • Restricting access to unwanted applications

  • Show windows version on desktop

  • Settings related to windows explorer, task-bar and start menu with features like power button management which definitely comes in handy.

  • Control over visual effects and animation.

  • Restart desktop and task bar automatically after errors

  • Automatic closure of non responsive applications with adjustable settings.

  • Speedup system shutdown

  • Optimization features like IDE auto detection disablement and disabling of windows hibernate function

  • Memory management : Ram and disk cache size selection, virtual memory and memory usage selection.

  • Settings related to context menu.

  • Neatest of all features is the console built in that keeps up records on changes made.
