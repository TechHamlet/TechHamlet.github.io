Facebook : How to Make a Custom HTML Page in Your Facebook Fan Page

Feb 27th, 2010 | Comments

I think you have seen some fancy HTML pages created within Facebook Fan Pages! Its like people have created a small web site inside their fan page! So, didn’t you thought of making a thing like that in your fan page? If not… think now…

Today I’m going to show you how to make a custom HTML page inside your fan page! As I said earlier you would have seen fancy HTML pages in some FB fan pages like :

So, do you want to make one like that on your fan page? Well, its easy. These codings are done using a Facebook Application called Static FBML.

Static FBML adds advanced functionality to your Page. This application will add a box to your Page in which you can render HTML or FBML (Facebook Markup Language) for enhanced Page customization.

How to Use Static FBML :

  • First go to the static FBML page, and select “Add to my page” :

  • Next, select the page you want to add static FBML and click “Add to page”

  • Now go to your page and select “Edit page”

  • In the edit page, under applications, you’ll find “Connect With US – FBML”. In there, click “Edit”

  • Now, give a title for your page in “Box Title” field. After that, give your HTML or FBML (Facebook Markup Language) code inside the “FBML” field.

Note : Do not give , and  tags! Think that your inside the tag!

Did you managed to build a custom HTML page? If so, feel free to leave the link as a comment!…. Enjoy!
