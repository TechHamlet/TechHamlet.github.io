Pokki : Bring a Cool Start Menu to Windows 8 and Add Great Web Apps

Dec 4th, 2012 | Comments

They first introduced Pokki as a platform for web apps. After you install it, you’ll be able to download and install many web apps from their repositories. They have apps for Gmail, Facebook, Instagram, Wunderlist, a lot of Games and for many more useful web services. Its easy and simple. Just like the Linux Software Centers and the Windows 8 App Store. But with the release of Windows 8, Pokki added a new great feature…

A cool new start menu for Windows 8. A big drawback in Windows 8 is the lack of the old start menu. That’s where we all start. But Windows 8 have replaced it with the new modern UI which is kind of a headache for the desktop users. To solve this problem, many developers have released little apps which can bring the old start menu back. Most of them have the exact same look of the old menu. Pokki will also bring back the start menu but it will have a brand new look…

It’s really cool. The new Pokki start menu have space for both normal Windows software and for Pokki web apps. You can set the favorite apps which you will see in the first menu. The right section will be dedicated to notifications. Mostly they carry notifications from the web apps.

If you right-click the Pokki start button, you can go to its settings. In this settings, page, you can change some Windows 8 UI settings like the location of notification banner, disable hot corners, settings related to the modern UI and it will also allow you to set the system to boot right into the desktop and skip the modern UI! A very useful feature for the new OS!

The app is useful even for older Windows users because of the web apps. But for Windows 8 users, it might be essential. some people will love the features offered by Pokki and its sleek look. So, what do you think? Like it?
