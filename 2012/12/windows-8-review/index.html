Windows 8 : The Full Review

Dec 6th, 2012 | Comments

Microsoft’s new Windows 8 operating systems is not only beautiful to look at, but it loads quickly and is touch-friendly, giving PC users like me a whole new experience when it comes to using my desktop, laptop and tablet. Did the new start screen catch your attention, as it did mine? It’s all new—and Microsoft’s trademark start menu has disappeared—with a tile layout that shows all of your computer’s content in one place. When I use my touch-screen tablet or ultrabook, I’m just a few taps of my fingers away from having the documents I need. The colorful tiles and applications can be moved around to create a customized screen that is organized just how I want it.

Another benefit of Windows 8 is, as I mentioned before, that it’s optimized for touch, which is great for those of us who have touch-screen devices. The interface is intuitive and makes it easy to swipe, tap and drag the components around as needed. With the touch keyboard and mouse features, it’s easy to type wherever we are. The keyboard has two modes, making it possible for us to get the most out of our portable devices when we’re on the go.

With this new release, we can get all of our work done on just one device thanks to the productivity features that make it easier for people who work remotely or on the road. We can also use it for our companies, as the enhanced security features such as Trusted Boot and BitLocker secure and protect the clients that are on our devices. This OS has built-in and cloud-connected applications that make getting work done faster and easier, as well as give me access to my favorite social media networks, games and media in my free time.

Windows 8 comes already loaded with the Calendar, Photos, Microsoft Mail, SkyDrive and People and Messaging apps that will automatically synchronize with my accounts on Twitter, LinkedIn and Facebook. With the Windows Live Syncing, and my Live ID, I can get to my documents no matter where I am. All I have to do it sign in to any PC that has Windows 8 and my settings and applications will appear instantly. The Windows To Go feature allows all of my settings, files and apps to be saved on a small thumb drive which can also be used on any PC on which Windows 8 is already installed. This makes it easy to go to from one computer to the next in work situations, or when it’s time to get a new computer, this will simplify the migration process.

SkyDrive lets allows my photos, documents, calendars and files to be stored in the cloud for easy access, no matter where I am. I will no longer need to take multiple devices with me when I travel for work, or waste time emailing important documents to myself just to make sure that I have access to them when I need them the most. Now, they are all in the cloud waiting for me.

It’s easy to get to the Windows Store from the desktop interface. The store contains thousands of applications which are grouped according to their category, so it’s possible to find the app that I’m looking for in a matter of minutes. Internet Explorer 10 is updated with a new look. There’s less frame and browser space, with more room to view the web pages. It’s also faster and more fluid, just like all of the other features in the system.

Getting started and learning my way around Windows 8 was simple, thanks to the pre-installed Getting Started with Windows 8 application. I only needed to launch it and read through the information which outlined all of the features of the OS simply and completely. This new version is unlike any previous one and offers the most intuitive, user-friendly interface to date. Even better is that it boots up in no time and maintains its fast speed all day long.
