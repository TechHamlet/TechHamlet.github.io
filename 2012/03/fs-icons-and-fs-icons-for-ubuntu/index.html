FS Icons and FS Icons for Ubuntu : Really Cool for Any Linux Desktop!

Mar 14th, 2012 | Comments

You can say that FS icons (Right) are cool but there’s nothing special about it. But you can’t say that to FS Ubuntu icons (Left). They match the default theme of Ubuntu. The colors and the design match perfectly. But have a closer look…

These new icons are really neat and elegant. If Ubuntu can use them as their default icon theme, then that will be a real wise decision. Currently, the FS Ubuntu Icons only have folder icons. But they plan to add more and even change the design a bit!

The next version of this icon pack, there will be a redesign, fleeing from the similarity of the YellowIcons icons, but also keep this version for download.

So, what do you think? Isn’t it worth checking? Try it out and let me know what you think!

Download :

  • FS Icons Ubuntu

  • FS Icons

How to Install :

  • After downloading, extract the archives. FS icons have to be extracted twice. The main archive have several more for individual icon sets. FS Icons Ubuntu also have to be extracted twice…

  • After that, copy the folders to /usr/share/icons (You need super user privileges to do this. Use the command “sudo nautilus” to open the file browser as root)
