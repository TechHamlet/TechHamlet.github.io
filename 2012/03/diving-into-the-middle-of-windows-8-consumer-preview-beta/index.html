Diving Into the Middle of Windows 8 Consumer Preview (Beta)

Mar 1st, 2012 | Comments

Yesterday night, Microsoft released the Consumer Preview of Windows 8. As promised, this new release comes with a lot of new improvements than the previous developer preview… Yesterday I started downloading it and now I have it installed in my laptop. Now, lets see what we have here…

Fish in the boot screen…

Lets start from the boot screen. Windows 7 have a nice animation where four dots get connected to create the Windows logo. It’s a big change compared to the old one they used in Vista and XP. In Windows 8, it’s going to change again. Now, we see a logo with a fish! Windows have gone fishing?

Online sign in…

After the installation, normally, we have to create an account in Windows. In here, we have to create an online account with our email. Then, Windows will sync the settings and metro apps with this account. So, if you sign-up with this account into another Windows 8 machine, you’ll get your old settings and apps back. It’s just like signing into your Android device with your Google account. Remember, Windows is providing all these settings because 8 will also work on mobile devices like Tablets!

Metro UI and the Old Desktop…

When you start your machine, you’ll end up in Windows 8’s new Metro UI (as shown in the top image). This is something which is totally aimed at mobile devices. All the apps are organized in tiles. You need to get used to it because even though they have the old desktop, this is the start menu. There is no start menu in the desktop.

Metro apps…

In Windows 8, you have two types of apps. The old kind of apps which we install and run in our desktop and the new ones which are called Metro Apps. They have their own UI. A different kind of buttons, menus, icons… Simply, those apps were to work in both desktop and mobile environments. To tell you the truth, they are really cool! For example, check out this News republic app :

But there’s one problem… There’s no close, minimize or maximize buttons! You have to use Alt+Tab or Window key + Tab to switch between apps. If you want to close them, you have to press Alt + F4 or they’ll be still running when you start your machine next time! How can it be? As we discussed earlier, Windows 8 won’t completely shutdown your machine. It will save some of the files in the memory to the hard drive and then turn it off. When you start, it will load faster than before by loading those files into the memory. It’s a half hibernate half shutdown. If you want to shut it down completely, go to “Control Panel > Hardware and Sound > Power Options > Choose what power button does” and untick “Turn on fast startup” (To untick this, you have to click on “Change settings that are currently unavailable” at the top). If you can’t find the control panel, simply press the “Window” key and go to the start menu. Now, start typing “Control Panel” and you’ll get what you want… Search any file, app or a setting in this way!

The App Store…

In the previous developer preview, Windows had a tile for the App Store but it didn’t work. Now they have their app store working on the consumer preview. If you are a Linux user, you might be familiar with this. One single place to choose,install,update and remove apps. Android and iOS also have this feature. I’m not sure whether Microsoft copied it from Linux or did it to make things easier in mobile devices. But if im correct, Linux used this concept before anyone else and it make things a lot easier and secure.

Some default Metro Apps…

Ah, Internet Explorer 10 is now available as a Metro App… Earlier, Google Chrome started a revolution by taking off the title bar and providing much space as possible for browsing. IE 10 Metro App have gone to the limits. You’ll get the address bar at the bottom and when the page is loading, the address bar will auto hide giving all the space to the website. You’ll see only a faint scroll bar. If you move your mouse to the left and right corners, you’ll see a faint forward and backward buttons. A completely new UI! To get the address bar, simply right-click on the site. But still, as a web developer, I hate Internet Explorer!

Now Windows 8 comes with a cool default Mail App. Since I logged in to Windows using my Gmail address as the live ID, the mail app automatically picked it as the default account. The best part is, it support Gmail without any extra configurations. I just have to give the mail address and the password! It looks like Microsoft is not that evil after all…

There are more very useful apps coming installed by default with 8. For example, now you don’t have to search for PDF readers… Windows 8 have its own reader which can be used as a great distraction free e-book reader! I really like that app…

The Ribbon…

Remember Microsoft introduced a new menu system called Ribbon in Office? Well, now its there in Windows 8’s Explorer. Rather than the traditional menus, Ribbon can make things easier to find and it organize things better. I didn’t like it in Office because it’s so big. But in Windows 8’s explorer, its collapsed. It will expand only when you click on it.

New file copy…

As we already know, the Windows file copy have also improved. Now its a bit faster than before on my laptop. But when I test the developer preview on my PC, it performed really fast! That’s not all, now you can even pause and resume the file copying process. If they can also add a feature to check the copied files for with the original file to see whether it have been copied correctly, I can safely forget about Terra Copy!

That’s only a small summary compared to all the new features which they have introduced in Windows 8. You must try it out yourself to feel the power. But do not replace your existing OS with it and use this as the primary option. This is a beta version so things can break! Try in a virtual machine or install alongside your existing OS.

Now there’s a battle going on. Ubuntu also have a lot of new features ready for their upcoming release. They also did a big change to their UI earlier. They also have plans for mobile devices… So, at the end, who will win?

Once, Linus Torvalds said that Gnome 3 had too many “Scary changes”. One can say the same thing about Windows 8. Its going to change how you work with the OS. A huge update! Will the users accept it? What do you say? Leave a comment and let us know what you think about this new OS!
