Want to Be a Real Linux Expert? Get Your Hands Dirty With Arch Linux!

Mar 4th, 2012 | Comments

Most of our readers are Ubuntu fans. But, some of them came to the Linux world in order to become real experts. Real geeks… But, if you think you are a geek just because you can move around Ubuntu, then you are completely wrong! (Im not talking about the experts who still love Ubuntu!). Ubuntu is built for beginners. Moving around Ubuntu is easier than most of the other Linux distributions. But, still its really powerful and even web servers use it!

But, if you really want to get your hands dirty and feel the full power of Linux, you better step out and look at yourÂ neighborhood. See what else is there. There’s Fedora, OpenSUSE, Debian, Slackware and hundreds of more flavours for you to try out. Fedora and OpenSUSE normally comes with a GUI and a set of packages for you to start with. Like Ubuntu, you don’t have to know much to start with these distributions. They have most of the things you need.

But, if you are already familiar with one of these distributions, then its time to try something hard…

Recently, one of my friends told me that I should try out Arch Linux. So, today I read about it before jumping in. Actually, it seems to be a really cool distro. Arch Linux is totally different from Ubuntu. It doesn’t come with a GUI and doesn’t have much packages installed by default (They have a lot in their repositories). It comes with only the base system. Then, you have to install what you want. If you want, you can install Gnome or any other desktop you prefer. You can customize everything you want. Now that’s really cool and it’s really good if you want to learn…

Another great thing about Arch is, it try its best to put the latest versions of packages into their repos. Ubuntu take some time to put new versions because they test them throughly. Debian takes a very long time and we usually get out of date packages. But people can argue that their system is good for stability and security. People can also argue that new versions of software can fix stability and security issues. For arch, they prefer latest software!

I think, if you start using Arch and if you survive to tell the tale, then you are a real Linux user. I’m saying this because you have to build your system. You have to configure it by your self. Don’t worry, when you get used to it, I’m sure that you’ll forget about GUI tools and stick with Linux configuration files. Its very easy when you get used to it. Give you more control over the system…

If you really want to get your hands dirty…

  • First check their beginner’s installation guide and install it in a virtual machine or as a second option in your machine

  • Then, check the more advanced Official Installation Guide and install it again

  • Then, check their Wiki and start building your system. They have a really good documentation. So, you should be able to learn everything very easily

If you are not satisfied, install it again until you get used to it. Then you’ll start to like Linux. You’ll want more. Then try SlackwareÂ or Gentoo. You need to try several distributions if you want to learn Linux.

Just now I’m also installing Arch for the first time. If things go well, I’ll write a detailed tutorial on how to get started. Then I’ll keep writing more Arch tutorials if you keep following them! So, don’t you want to be an expert? Don’t you want to get your hands dirty? Try Arch and let us know what you think about it…
