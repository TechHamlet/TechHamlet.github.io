Ubuntu 12.04 Runs on New Laptops Without Over Heating!

May 3rd, 2012 | Comments

It’s a known fact that we can’t install Linux in most of the new laptops without breaking them. Many machines start to over heat because the Linux kernel turns off some power saving features of the CPU. Mostly, the new Core i processors face this problem. According to Linus, the problem will be fixed in the upcoming Linux kernel 3.4.

Don’t worry, Ubuntu 12.04’s kernel (Kernel 3.2) have the patch to fix this problem. So, you can safely install the new Ubuntu 12.04 on your laptop without melting it. I installed it on a Core i7 lap and the CPU heat is between 52 to 55 Celsius. (Im living in Sri Lanka and its a bit hot these days). In Windows, I get a temperature around 54 to 57. Better than Windows!

You can solve this problem by turning off the main GPU and switching to Intel HD graphics. We’ll see how to do this in the next post! So, stay tuned!
