How to Install Latest Nvidia Drivers in Ubuntu for Maximum Performance

Nov 23rd, 2012 | Comments

Image Source : Stanford EdTech

Currently, the official Ubuntu repository have several versions of Nvidia drivers. By using additional drivers, we normally install two versions. The current (well-tested) drivers and the current-updates drivers. Both of them are pretty old. Nvidia have recently released a huge number of updates to their drivers which really increase the performance by great amounts!

Earlier, Nvidia didn’t pay much attention to Linux. But… Two things changed their minds. First… In a recent event, the father of Linux Linus Torvalds criticized Nvidia for their poor Linux support. He also showed ‘the finger’ to Nvidia in this event! He openly offend people who need to be offended. Most of the time, he is right because most of his victims accept their mistakes and fix their issues! Nvidia have also accepted their mistakes.

The next thing that changed Nvidia’s mind is Windows 8! With the release of Windows 8, many game developers including the creators of the most popular gaming platform Steam called it a “Gaming Catastrophe”. So they started to test Linux versions of their software and games. To support these games, Linux need to have a more powerful graphic drivers. So, Nvidia thought of updating their drivers!

Currently, the official Ubuntu repository have the Nvidia driver version 304 which is old and doesn’t have enough performance for games. But that drivers is the most stable because they are very well-tested by the Ubuntu developers. The latest Nvidia driver 310 is also available in the repository. But they are tagged “Experimental” and haven’t been tested.

Today, I just installed these experimental drivers and to tell you the truth, I did feel a difference in performance. The animations are now smoother. Sometimes, these experimental drivers can cause issues because they are not tested. But still I haven’t faced any issues… So, if you want, you also can try it :
