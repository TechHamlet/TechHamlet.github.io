Ge.tt : A Fast and Easy Way to Share Files Without ADS!

Aug 28th, 2012 | Comments

If you need to share a file to your friend, if you need to do it quick, if you don’t want to sign-up to any file sharing sites to do it, if you need fast and real-time access to the files, where will you go? There are many services you can use… Send Space, Rapid Share, Hotfile, etc… You can upload and share a file  using these services without signing up for an account. Its fast and easy. But there’s a problem… A big problem…

These websites are full of ads. For example, if you upload a file to Send Space and send that link to your friend, he might get lost. In their download page, the real download link is small but there’s a big ad with the title Download in it. Most probably, your friend will click the ad and start cursing you!

But there’s one service which we can use without any fear! Ge.tt…! Just like the name, the service is really simple and easy to use. The best part is, there are no Pesky Ads!!! No need to sing into any accounts. Just go to the site, upload a file, and share the link with your friend. Or, you can type his email address and send an email right from the upload page. Not just one, you can upload any number of files and share them with one single link. Maximum allowed size for a single file is 2GB! That’s very big compared to the competition.

Ge.tt

Share

What will your friend see? A simple page with some details (The size, upload date and the number of downloads) about the upload and clearly visible real download links. If it’s an image, he’ll see a preview. Simple, minimalistic but have what we want.

Download

According to Ge.tt, all files you upload (without signing up for an account) will be deleted automatically after 30 days. This is ok because you wanted a quick real-time sharing service.

If you create an account, you’ll get 2GB of storage space for free. These files won’t be deleted automatically. If you want more, you have to pay more. But, Before you upload anything, read this :

This is normal in file sharing services. There are some people who encrypt our data and keep them secure and inaccessible to others. But most of the people doesn’t do that. For the purpose of Ge.tt, we don’t need it much. We need a way to instantly share a file. Not to store sensitive data for a long time.

If we compare with other instant file sharing services (that I know), I’ll put Ge.tt to the top of the list because its minimal, nice, easy to use, fast and most importantly, free of those pesky ADS!
