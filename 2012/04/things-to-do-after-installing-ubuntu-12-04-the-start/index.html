Things to Do After Installing Ubuntu 12.04 : The Start

Apr 28th, 2012 | Comments

Most probably, you might have already installed the latest Ubuntu 12.04 in your machines. But, if you are new, you might be thinking what to do next! If you are really new (coming from Windows), you might be totally losst. Don’t worry, here’s some tips to start your journey.

Note : I’m writing this article for everyone including extreme beginners. So I’ll try to point out every detail. But, im going to use the Terminal to do most of our work. If you get used to it, it’s the best way to move around Linux!

  • Before we begin, I think it’s a good idea to add the “Terminal” into your launcher (The bar on the left side of the screen.). To do that, click on the “Dash Home” button (located at the top left corner) and type Terminal.

  • Now click on the Terminal icon. Then you’ll see its icon appear on the left Unity launcher. Right click it and select lock to launcher. Now you don’t have to go to the Dash. Even after you close the terminal, you’ll find that icon in the launcher.

First of all, lets put up the defenses…

When it comes to Linux, most of us think its indestructible. Many think that no virus can harm it. You are wrong… Linux have a good security system but that doesn’t mean its 100% safe. People thought the same thing about Mac. Now, thousands of Macs are infected by the deadly Flashback trojan. So, security first…

Note : In Windows, we have Administrative users and normal users. If we have a normal user account, we can’t do much with it. We can’t even install any software with it. Some consider using it than the Administrative user account because if you’re an Admin ,you can install anything. That means, malware can also get installed on to your machine through your account. In Windows Vista and 7, the system warns us when installing software or when they change any system files. In Linux, we always get a normal user account. Its strict. If we want to install something or change system settings, always we have to give our password. When it comes to activating and de-activating the firewall also, only the admin user (in Linux we call him the root) have the right to do it. So, if the normal user want to get the amin privileges temporarily, we use the sudo (super user (that’s the root) do) command.

Next, activate the Canonical partner repositories and update the system…

Ubuntu stores all of its software in online repositories. The Canonical partner repositories contain some useful software like Adobe Flash. But it’s not enabled by default because they are not open source. But its a good idea to enable it. There’s an easy way to do this through the graphical tools. But, today we do it the hard way!

Next, install the restricted drivers…

Normally, Ubuntu should detect all your drivers and install them automatically. When it comes to drivers like the graphic, Ubuntu installs the open source drivers by default. Some of these drivers does not support features like 3D acceleration. So, to enable them, you have to install the proprietary drivers.

  • Now, open the dash and search for Additional Drivers and launch it. In there, select the driver you want (Select the latest drivers) and select Activate at the bottom right corner. Note : If you have a laptop with switchable graphics, do not do this! We’ll talk on how to make things work for those systems soon.

Now, lets adjust the privacy settings…

Starting from Ubuntu 12.04, you can adjust the OS privacy settings from the settings window. Normally, Ubuntu records your activity. Notice how the dash display your recently used apps and recently accessed files? If you think its breaching your privacy, you can turn them off.

  • Again open the dash and type Privacy and launch it. You can use these settings to control how Ubuntu tracks you and delete the history if you want.

Ok, this is enough as a start. Now you have setup your system and you are ready to go. Now, explore the new OS. If you are coming from Windows, and completely new to Ubuntu, you might think that the look of the OS is boring. Don’t worry, we can customize every piece of the interface and we’ll teach you how to do it in  the upcoming tutorials. If you are curious, try to install some apps like My Unity and Ubuntu Tweak. Try to figure out how to do it. Anyway, we’ll talk about them next.

If you have a laptop with switchable graphics, you’ll notice that the heat is rising. The CPU heat is also rising but it’s not the fault of the CPU. Earlier, Linux overheated new laptops (specially the new core i* based laps). But Ubuntu 12.04 have completely solved that problem. I have no heating issue in my Core i7 lap. Earlier it became so hot in just a few minutes. The problem now lies with the switchable graphics. If you are a pro, try to disable the main VGA and enable the Intel HD graphics using VGA switcheroo. This should fix everything. If you don’t know what im talking about, wait! I’ll tell you how to do it very soon! Now, start exploring!
