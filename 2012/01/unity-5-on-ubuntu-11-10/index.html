Install the Improved Unity 5.0 on Ubuntu 11.10 [Faster Than Before]

Jan 14th, 2012 | Comments

The Ubuntu developers have recently updated the development version of Unity to version 5.0 which will be available on Ubuntu 12.04. This new Unity seems to be faster and more stable. The team have listened to the users and have made several improvements and have added some useful features into their official desktop. So, don’t you want to try it out?

One major problem we had in Unity is the slow responsiveness. If we want to launch an app, if we want to search for something in the dash, Unity always take a lot of time. It’s like its getting stuck for a second! But the new Unity is really fast. Think its faster than the Gnome Shell! The main target of the developers is to make the UI more stable and faster.I think they have done it well.

Earlier, some people had some difficulties finding the app menu because it disappears just after you launch a software. But now, it will be there for several seconds before disappearing. You can always make it stay longer by changing the settings.

Now you can close an open app by dragging its icon on the  launcher to the trash bin. I think you can also use this feature to get rid of frozen apps! Not only that, now you can jump to the lens you want by just right clicking the dash icon.
