A Siri Like App for Windows!

Jan 3rd, 2012 | Comments

iPhone 4S became very famous due to the arrival of the voice assistant Siri. Lots and lots of programmers started to create similar applications on other platforms. But almost all of them were not not much efficient as Siri, as Siri was created by the experts of Apple. And here is the first Siri like application for windows. Lots and lots of Siri like apps came out for android and even for windows mobile but not for Windows. So here is the first Siri like application created for windows. I found out this application when I was stumbling over the web.

This application is named as Mitini which was created by an individual which is really a appreciable task. This application is also not as much efficient as the Siri for Apple. Even the size of this application is reduced to just 256 KB ! And so now lets take a look at it

Mitini For Windows :

  • This application as I said before is not as much efficient as the Siri for apple and this application does not recognizes your speech instead you need to type in what you are about to ask.

  • It does not understand what ever you type. Some times it turns out to say “Sorry I Dint Get That”

  • It is actually your text assistant.

  • The results take some time to return, but Siri answers you instantly.
