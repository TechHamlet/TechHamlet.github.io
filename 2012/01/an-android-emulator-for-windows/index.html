Jan 10th, 2012 | Comments

Android is currently the best operating system for smart phones available out there. Most of the people here in India are longing to get their own Android including me. Today when I was stumbling over the internet I found this new application named as Blue Stacks which enables users to run android applications on their windows machine. This application is not much bigger by the size of it. It is just 117 MB which is a too low size for such a good application. This application comes with a few pre installed apps.
