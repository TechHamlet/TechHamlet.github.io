Update : Ice Cream Sandwich Gnome Shell Theme : A Linux Desktop Into an Android

Jan 29th, 2012 | Comments

If you are an Android fan, you can now take your Android system into your Linux desktop! Not the whole system, but you can take the look of the new Ice Cream Sandwich with this new Gnome Shell theme. You can see how cool it is by going to the applications section of the shell. A minimal theme with a bluish outline!

Unlike the two themes we introduced earlier, this theme doesn’t have a GTK theme. Only the shell theme. But the developer have promised to try building GTK2/3 themes. But he says it will look ugly!

I will also try to make a GTK 2/3 theme and some metacity borders… but it might be difficult and the result will be probably ugly

I tried it with the Faenza icon theme. But it might look better with the Faience icon set which is also developed by the same person! If you want to try it out :

  1. Download the theme from here : http://tiheum.deviantart.com/art/Gnome-Shell-Ice-Crean-Sandwich-280076980

  2. Follow the instructions in my previous post to install the theme : Zukini and Zukitwo : Two Awesome GTK2, GTK3 and GNOME-Shell Themes

Update : Many people have asked whether there’s a good GTK theme to use with this one… Recently, the developer have updated the theme and now it contains a matching GTK theme! (Note : The theme developer have just renamed the theme to HOLO) WebUpd8 have just uploaded the theme files into their repository… You can easily download and install the theme from there : http://www.webupd8.org/2012/01/holo-android-ice-cream-sandwich-like.html
