Change Ubuntu User Passwords Quickly and Easily

Jan 23rd, 2012 | Comments

If you have forgotten your Ubuntu operating system password, you can’t install new software and in some situations you can’t even log-in to your account. However now you need not afraid, we can easily change it. For this process we do not need complicated Linux commands. Just in simple three steps, we can change all the user account passwords. If you have more than one account then, need to change them separately, currently there are no bulk changes.

First reboot or boot your computer, if you installed Ubuntu inside the windows OS,then select Ubuntu in Windows boot manager using arrow keys and press enter key to go forward.

In the Ubuntu’s boot menu, select “Recovery mode” and press enter key on your keyboard. Now its boot process start, wait for a few seconds to complete this process. After finished, you can see “root@ubuntu:” command now type the following command in there,

Ex: – passwd home

__ Now type your new password in “Enter new UNIX Password” field and press enter key, again type it for confirming. If you successfully complete this task you will receive “passwd: Password Update successfully”. If you have more than one user account then, again type “passwd ” and type the new password.
