Five Important Steps to Secure Your Word Press

Feb 20th, 2012 | Comments

Website security should always be the highest priority of any webmaster or blogger on WordPress.  This web blog site proves more susceptible to vulnerabilities than any other blog hosted service available.  WordPress is one of the most frequently used CMSs and is often the target for hacker attacks.  The more traffic you get to your blog, the higher risk you become of being attacked by hackers.

There are various methods that should be applied to ensure the security of your WordPress blogs. Some of these methods prove straightforward with only taking a little time. 

The following steps can help increase the security of your WordPress blog website:

Step 1

Continually check for updates

WordPress is continually updated regularly with security improvements continually occurring.  No system is ever one-hundred percent secure, which is why you need to keep your WP version updated regularly to get the advanced security features.  It does not prove difficult to update your website.  Take a look at the Admin Dashboard in order to see the latest updates.  You will receive messages informing you that updates are available.

Step 2

Back up your WordPress installation

All webmasters and bloggers understand the importance of backing up their installation.  Sadly, only a minority regularly backup their installation with the majority occasionally performing a backup installation. WordPress contains an export function for posts of the blog themselves.  Regardless, it proves prudent to create your own backup into your WordPress folder.  Solely relying on the hosting company to back up your website is never enough.

Recommended backup plugin:

WP-DB-Backup – This plugin lets you effortlessly backup your WordPress database tables.  You can backup, other tables in the database, as well.

Install

  1. Extract the wp-db-backup/folder file to/wp-content/plugins

  2. Activate at your blog’s Admin –>Plugins screen

  3. WP-DB-Backup will try to get a directory /wp-content/backup-*/ inside your WordPress directory.

  4. You might want to make /wp-content writable (at least temporarily) for a list to be created.

Step 3

Get rid of the default Admin Username

Never keep the admin name as “admin”, because this will increase the chances of hackers attacking your WordPress website.

You can perform one or two options in keeping hackers at bay from your website:

  • You can create a new user for admin and relegate the admin account to subscriber only. This will lead hackers to believe that the admin account exists, and they cannot make changes to your blogs.

  • Delete or change the admin name

Step 4

Use WordPress security plugins

There are various selections to choose from when seeking WordPress security plugins.  Installing plugins can help in making your WordPress web blogs more secure than ever.

These include some the following:

WP Security Scan – This plugin consistently checks your WordPress blogs and website for vulnerabilities while making recommendations to fix them.

WP Security Scan can check your website/blog for vulnerabilities while making suggestions for corrective actions.

It also performs the following features:

  • Version hiding

  • Database security

  • Passwords

  • File permissions

  • WordPress admin security and safety

  • Removes the WP Generator META title from the core code

 Install

  • Create a backup of the current installation

  • Unpack the download package

  • Upload extracted data to the /wp-content/plugins/ directory

  • Locate the “Plugins” in WordPress and activate

Lockdown plugin– (Offered by default by WordPress) “Lockdown” records IP addresses and timestamps of each failed login attempt. If constant attempts are detected within a small period from the same IP range, the login function will disable itself from that range. This proves useful to prevent hackers from discovering your password.

Install

  • Extract the zip file into the “Plugins” directory into a separate folder.

  • Activate the plugin from the Plugin options

  • Customize settings from Options panel

  • Extract the zip file into your plugins index into its own folder.

  • Activate from the Plugin options. .

  • Customize settings from Options panel.

5. Create strong and secure passwords

The one thing you should keep in mind when creating passwords is to never input birthdays or your name as your password. Hackers or anyone with half a brain can figure out these types of passwords. Instead, select small letters, punctuations, numbers and capital letters in creating strong and secure passwords.

The tips on securing your WordPress blog written in this article are a few steps that show how you can accomplish this task.  You can always find other methods, but the five steps mentioned in this article are the basics and should never be disregarded.  Despite how unsophisticated and trivial these tips are, you will find they work exceptionally in securing your WordPress website and web blogs.
