I Just Showed Windows 8 to a Friend : "Oh.. This Is Not User Friendly"

Jun 8th, 2013 | Comments

When Microsoft first released Windows 8, I had my own doubts. I love new stuff but can I get used to Windows 8? I used it and thought I might be able to get used to it. I thought I liked it. But that’s not the truth! I found out how I really felt about Windows 8 when I showed it to a friend today. As soon as he saw the new start screen, he saidÂ ”Oh.. This is Not User Friendly. Windows 7 is way better!“…

If I get a response like this from a novice user, it won’t be a problem. It’s really hard to find your way through Windows 8. But my friend is an Engineering student. He’s not into IT much but still he know his stuff. He work with the computer everyday. He said its a mess. Full screen apps are ugly for him and problematic. Desktop without the start button is also a problem to him (Recently, Microsoft have confirmed that Windows 8.1 will have this start button back). I didn’t told him that Windows 8 is also built for tablets. But he immediately said “This is built like a tablet!”.

Windows 8

It also became hard for him to find his way through Windows 8…

Earlier we posted a video in our forum showing how a group of people presented Windows 8 to a novice user and asked her to shut it down. It took a very long time for this novice user to find the shutdown button. When introducing Windows 8 to my friend, I showed him that there’s a black bar that can be brought up by moving the mouse to the bottom right corner. I told him it have the shutdown option. But when I gave him my laptop, he again asked me where is the shutdown button? When I looked at the machine, he was pointing the mouse to the bottom right corner but there is no black bar! For a second, I was also stunned. Then when I did it my self I realized that we have to move the mouse to the corner and drag it upwards in-order to get the black bar. I’m used to it so I didn’t realize the problem. My friend used the new OS for some minutes and kept it aside saying “Windows 7 is way better!”.

I realized that I also don’t like the new Windows 8 that much…

I thought I liked the new Windows 8. But I was wrong. When my friend asked me how I managed to live with it, I said, I’m not using it that much! That’s not my main OS! When Microsoft released Windows 7, I was using Linux as the main choise. But Windows 7 made me switch. I picked Windows 7 as my main OS. But after several years, I went back to Linux. With the release of Windows 8, again I switched and this time I was confident that I will make it my main OS. But actually, I didn’t use it for long. I came back to Linux! Later, my Linux installation got extremely buggy and crashed all the time. Usually, I move to Windows in a time like this. But this time, I stayed with Linux.

Earlier I didn’t realize that I don’t like Windows 8 much. When thinking about my friend’s review on 8, I started to think about what I really think about 8. Recently, people started to dump BIOS in favor of UEFI. I upgraded my PC and got a new Laptop. Both have UEFI now. In these systems, If I install Linux, it won’t bring up the boot loader (once I wrote a post on how to bring back the boot-loader and add Windows 8). Once I fixed this because I thought I wanted to switch operating systems daily like I did before. But later I ignored this. Of -course I can use the UEFI interface to switch OS. But its not easy. I had to press Esc or F10 every-time the system boot. If I mess up the timing, I have to reboot! But why did I ignored the boot loader issue? I didn’t feel like going back to Windows. I had everything I wanted in Linux!

It’s not hating Windows 8… It’s because there are better options than 8. For my friend, he have Windows 7. For me, I have Linux! Can Windows 8 beat these two? I don’t think so! What do you think?
