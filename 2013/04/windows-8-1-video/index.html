Windows 8.1 : Whats New in the Leaked Build 9369 : Video

Apr 17th, 2013 | Comments

Recently, a new build of the upcoming Windows 8.1 have been leaked to the web. Many people have already tried it and now WinBeta have created a video walk through explaining its new features.

The latest build of Windows 8.1 have a new Files App for the metro UI. It’s the metro version of Windows Explorer. It also have a new PC Settings pane with more new sections and a much cleaner look.

There are no major changes like the return of the start button and the boot to desktop options. But still Windows 8.1 upgrade will make Windows 8 a bit easier to use. It looks like they are porting more and more features from the desktop version to the metro version. Are they trying to get rid of the desktop version after they take everything to the metro ui? Recently, a Microsoft Exec have claimed that they have no intention to get rid of the desktop. So, we don’t have to worry about it yet.
