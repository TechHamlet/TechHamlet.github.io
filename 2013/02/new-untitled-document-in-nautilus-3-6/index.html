How to Create a New Untitled Document in Nautilus 3.6? (Ubuntu Linux)

Feb 13th, 2013 | Comments

Last time I showed you how to install Nautilus 3.6 in Ubuntu and Linux Mint. In Linux Mint, it won’t replace the default file manager because Mint use Nemo. So, there won’t be any issues. But in Ubuntu, since they use an old version of Nautilus, the new one will replace it. When that happens, it might give out some issues. For example, now you can’t create a new Untitled Document in Nautilus 3.6 !

Now this is the reason why Ubuntu didn’t update to Nautilus 3.6… The new file manager have made some big changes in-order to fit it perfectly with Gnome Shell. But these changes will break the system under Unity. So, Ubuntu sticked with the old version. That didn’t stop some of us from using Nautilus 3.6 because some of the new features are really great! But, you won’t be able to create a new Untitled Document in Nautilus 3.6 using the right-click context menu. So…

How To add the ability to create a new Untitled Document in Nautilus 3.6?

The answer is very simple… You just have to create a blank document in the Templates folder…

  • Go to your Home folder (/home/your-user-name)

  • Now, open the Templates folder

  • Next, open your text editor (Gedit or any other editor that you prefer) and save a new blank file in this folder

Templates Folder

Now, you should get the create a new Untitled Document in Nautilus 3.6’s right-click context menu. Not only untitled documents… You can add any type of document to this folder including any of the Libre Office formats, HTML, CSS, TXT, RTF, etc. Read the text that appear on the top when you visit that folder. It explains the functionality of Templates. “Files in this folder will appear in the Create Document menu”. Problem solved? Good!
