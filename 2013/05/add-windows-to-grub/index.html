Add Windows to Grub Boot Loader : Linux With UEFI and Windows 8

May 17th, 2013 | Comments

At the moment, people are moving away from the old BIOS system to the new UEFI firmware. If its done correctly, it will be much easier to dual-boot multiple operating systems. But unfortunately, Microsoft and some hardware vendors try to break the UEFI system and tie us to Windows 8. But in our previous posts, we saw how not to get caught to their tricks and pick the best machine. Then we saw how to get back the boot loader after installing Linux. But still, you won’t have Windows in it. So, here’s how to add Windows to Grub boot loader…

Because the UEFI firmware enables you to boot into multiple operating systems directly through its boot menu (people still call it the BIOS), Linux won’t show the Grub boot loader by default. Because of this, Linux doesn’t even add Window to Grub boot-loader. So, if you have a machine with both Linux and Windows, you might want to add it into the Grub boot menu instead of going to the UEFI system every time the machine boots. So here’s…

How To Add Windows to Grub :

Only for systems with UEFI enabled and with a GPT partition table. Use fdisk -l to learn whether you have a gpt disk. If the output doesn’t show GPT, then you are not using it!
