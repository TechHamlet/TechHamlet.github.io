Unity Tweak Tool : Tweak, Customize and Control Ubuntu From One App!

Jan 17th, 2013 | Comments

We have plenty of system utility suits, tweakers, cleaners and optimizer for Windows. For Linux, we don’t need to perform that many maintenance tasks and system tweaks in Linux. But still, if you are interested in diving deeper into the system settings, there are several apps you can try. The Unity Tweak Tool is one of them…

We have reviewed several Linux system tweakers… When Ubuntu introduced Unity, it didn’t offer much options to easily tweak its settings. So, the community came up with My Unity. The simple tool allowed us to tweak most of the settings in Unity. Later, Ubuntu added it to the official repositories but now, it have vanished!

Apart from My Unity, haven’t you heard about Mechanig? An old system tweaker for Ubuntu.It started as a small cleaning tool and got more functionality as it grew bigger. Recently, the developers have decided to change its name, the appearance and add some more cool features to Mechanig. So, they introduced the Unity Tweak Tool!

Unity-tweak-tool was born as Mechanig but was renamed twice to this very name. Reasons for the renaming varies from being too witty for the users to naming issues with the Unity developers.

- Freyja Development team

Don’t let the name fool you… It’s not just a tweak tool for Unity. It cover more areas in the Ubuntu system. First section will allow you to control most of the important settings in Unity. Its even more powerful than My Unity. The tool also have easy access to settings Earlier we used the more complex Compiz Config Settings Manager for this.

Unity

The windows manager section will also offer some advanced settings available in CCSM. You can even set the Hardware acceleration settings and animations through this section. Settings for Work-spaces, Window Spread, Window Snapping, Hot-corners are also available.

Window Manager

Appearance section is dedicated to the look and feel. You can change your theme, icons, fonts and some window control settings…

Appearance

The last System section have settings for desktop icons, scrolling and some small security tweaks….

System

Actually, the Unity Tweak Tool is still under development. So, more new features will be introduced in the upcoming days. You can try it out by adding the PPA. But remember, it will work only for Ubuntu 12.10 and 13.04! No support for Ubuntu derivatives like Linux Mint.
