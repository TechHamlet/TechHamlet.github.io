Why Haven't You Switched to HTML5 and CSS3? Scared?

Jan 10th, 2013 | Comments

Image Source : Randall Benson

If you are a web designer, you might have already heard about the latest web standards HTML5 and CSS3 (If not, you are not a web designer). When they introduced these standards, every one thought they can change the world with these new technologies. But many feared to use them on their designs because most of the browsers didn’t fully support them. But now, things have changed…

HTML5 introduced a more simple and a cleaner way to write code. It introduced some new tags to easily differentiate things in a website without using div tags for every single element. Most importantly, it  replaced Flash by introducing new standard ways to deal with multimedia. Now you don’t need any third-party plugins! CSS3 introduced animations! Earlier, we had to be JavaScript experts to use animations on our designs. But now, we can use CSS to do that job. We can create 3D objects, add shadows, add border radius, twist and turn an element to any shape we want… Because of this height flexibility, now we can even do a logo creation using CSS.

Back then, the browsers were really old…

When they first introduced these technologies, people were widely using old and outdated browsers. IE 6 was widely used. Most of these browsers didn’t even knew about these new technologies while many others have only partially implemented them. People using Google Chrome didn’t had much issues because that browser automatically update itself to new versions. But IE and Firefox were hard to update. We have to do it manually. So, people ignored them…

The other problem we faced is the slow development cycle of browsers. The developers didn’t update their browsers frequently. Some time it take years to get something new. So, they were very slow to adapt these new technologies. Because of these problems, many web designers refused to work on these new standards.

But now, The old browsers are dying fast…

Things have changed… IE 6 is now on the death-bed. Its dying out. Google Chrome became very popular because its development team was really fast. They released new versions very rapidly. So, that browser adapted these new technologies really fast. All the others followed it. Firefox also adapted a rapid development cycle. In a recent release, they also added automatic updates. So now, the platform is ready…

You just have to start…

If you are a designer who ignored HTML5 and CSS3, now its time to look at them again! I switched to it several months ago and I immediately felt how easy it is to switch. Things will become easy, more beautiful… You’ll have more freedom to put better designs. So, why wait? Switch today!

HTML5 and CSS3 didn’t just add more features to the web… They also heavily upgraded our browser market! Now, the developers will always keep their browsers up-to-date with the latest standards. So, in the future, you don’t have to wait!
