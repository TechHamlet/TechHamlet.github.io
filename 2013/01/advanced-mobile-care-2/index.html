Advanced Mobile Care 2 Is Here! (Android)

Jan 19th, 2013 | Comments

Last November, IObit released their first Android utility suite, the Advanced Mobile Care. It contained almost all the tools we want to keep a healthy device. A system tuner, Mobile anti-virus, Game speeder, Battery saver, App manager, Task killer, and a Privacy killer came with the initial release. But now, it just got some more cool features…

The Privacy Adviser is a very useful feature added to the new version of Advanced Mobile Care. When installing apps through Google Play, it will show you what the app can access on your phone. Some apps might ask for permissions that they never need. Some might be using those permissions for malicious activities. So, you always need to be careful about the permissions you grant. But if you ignored them, you can use this Privacy Adviser to review the permissions again and remove dangerous looking apps for good.

Privacy Adviser

Another new feature is the Cloud Backup. Now you don’t have to worry about loosing your important data like the contacts and the call logs. You can use the new Cloud Backup to store them in a safe location. You can also use this feature to move your data easily from the old device to the new one.

Cloud Backup

Normally, if you want to remove apps from Android, you have to do them one by one. It’s a painful task if you have a list to remove… Now you can use Advanced Mobile Care’s batch uninstaller to do it with one sweep! Things are getting easier…

The new Advanced Mobile Care 2 also have home screen widgets for easy access. You can use them to issue quick scans, activate the battery saver and get a quick look at the system information. Apart from those changes, there are bug fixes and small updates coming with this new version.

Widgets

If you already have the app, update it. If not, try it out! I don’t think you’ll refuse it.
